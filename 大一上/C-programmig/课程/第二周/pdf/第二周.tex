下面主要介绍一下位运算：

运用一：交换两个数
\begin{lstlisting}

int main(){
  int a,b,c;
  scanf("%d %d",&a,&b);
  a=a^b;
  b=a^b;
  a=a^b;

  printf("%d %d",a,b);
}
\end{lstlisting}

运用三：定义掩码

\noindent 对于0000我们有：\begin{itemize}
  \item \&同化为0
  \item |不做处理
  \item \^不做处理
\end{itemize}
与此同时，对于1111：\begin{itemize}
  \item |同化为1
  \item \&不做处理
  \item \^用于求反
\end{itemize}
三个功能，同化，通过，求反。
