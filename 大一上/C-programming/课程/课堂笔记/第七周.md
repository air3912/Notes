##### 递归求阶乘

```c
int function1(int n){
    if(n==1)return 1;//基本情况
    return n*function1(n-1);//递推关系
}
```

##### 跳阶梯

- 一只青蛙一次可以跳上1级台阶，也可以跳上2级。求该青蛙跳上一个n级的台阶总共有多少种跳法

```c
int function2(int n){
    if(n==1)return 1;
    if(n==0)return 1;
    return function2(n-1)+function2(n-2);
}//但是可以明显的感受到计算量，所以其实更推荐用动态规划
```

##### 汉诺塔

```c
void move(int n,char a,char b,char c){
    if(n==1){
        printf("将其移动至c即可")
    }
    move(n-1,a,c,b);
    printf("将%d从%c移动%c",n,a,b);
    move(n-1,c,b,a);
}//其实这边构造的时候建议不要看太多，就是只考虑n==1和n==2的时候是怎么样的去构造
```

#### 变量

- external 多用于声明（可以配合后置定义）

- static 静态变量，分为函数内和函数外，主要作用就是封装

#### 求最大公因数

```c
int gcb(int a,int b){
    while(b!=0){
        int reminder=a%b;
        a=b;
        b=reminder;
    }
    return a;
}
```
