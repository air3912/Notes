# [2018 Fall C Midterm] maximum_power

## Description
Given an integer $n$, find the maximal integer $x$ that satisfies $$2^x \leq  n$$

(求不超过正整数n的2的最大幂值。)

 

## Input Format

The integer n (1≤n≤100000)


## Output Format

The maximum power $2^x$

## Sample Input
```
9
```
## Sample Output
```
8
```
