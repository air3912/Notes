# 2021程序设计I 期中考试-A卷10-进制转换

## 题目描述

输入一个由 $x$ 进制描述的**正整数** $n$，将其转换为 $y$ 进制输出。

## 数据规模

所有数据均满足 $2 \leq x, y \leq 36$，即 $n$ 可以用阿拉伯数字和大写英文字母来表示，且 $0 < n < 2^{32}$。

## 输入格式
输入包含两行：
第一行包含两个用空格分隔十进制数 $x$ 和 $y$，分别表示 $n$ 的输入进制和目标进制；
第二行包含一个 $x$ 进制数 $n$，本行不含任何空格，以换行符 `\n` 结束。

## 输出格式
输出包含一行，为 $y$ 进制数 $n$，输出应当以换行符 `\n` 结束。

## 输入样例
```
15 17
1A2B3C

```

## 输出样例
```
F4879

```
