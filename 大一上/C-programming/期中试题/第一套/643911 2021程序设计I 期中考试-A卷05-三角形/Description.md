# 2021程序设计I 期中考试-A卷05-三角形

## 题目

输入一个正整数$n$， 打印一个等腰三角形，这个三角形有$n$行，第$i$ $(1\leq i \leq n)$行有$i$个字符，其中$i$为奇数时，该行为字符`*`，$i$为偶数时，该行为字符`#`。对于第$i$行第$j$列字符，如果$i+j$是$5$的倍数，则输出`5`，不考虑字符`*#`。
下面是$n=3$的例子：

```
*
##
*5*
```

## 数据规模

所有的数据均满足 $1\leq n \leq 30$。

## 输入格式

输入共有1行，一个整数$n$。

## 输出格式

见样例，**注意**不要输出行末空格。
请**注意**输出应以换行符`\n`结束。

## 输入样例

3

## 输出样例

```
*
##
*5*
```
