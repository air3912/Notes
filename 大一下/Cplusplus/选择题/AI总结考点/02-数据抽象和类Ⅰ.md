# 数据抽象和类Ⅰ：选择与判断题

> 覆盖第3、4周。单选题均只有一个正确答案。

## 一、单选题

1. 使用 `class` 定义类且未写访问说明符时，成员默认为：
   - A. `public`
   - B. `protected`
   - C. `private`
   - D. 由编译器决定

> 答案：**C**。

2. 使用 `struct` 定义类且未写访问说明符时，成员默认为：
   - A. `public`
   - B. `private`
   - C. `protected`
   - D. 不允许省略

> 答案：**A**。

3. 在类体内部直接给出函数体的成员函数：
   - A. 必须再写一次 `inline`
   - B. 隐式具有 `inline` 属性
   - C. 一定在运行时展开
   - D. 不能在类外调用

> 答案：**B**。 解析：`inline` 属性不保证编译器一定展开。

4. 下列默认实参声明合法的是：
   - A. `void f(int a=1, int b);`
   - B. `void f(int a, int b=2);`
   - C. `void f(int a=1, int b, int c=3);`
   - D. `void f(int a=1, int b, int c);`

> 答案：**B**。 解析：默认实参必须从右端连续出现。

5. 类中只声明了 `A(int x);`，则下列说法正确的是：
   - A. 编译器仍必然生成 `A()`
   - B. `A a;` 必然合法
   - C. 编译器不会因此自动补一个无参默认构造函数
   - D. `A(int)` 不是构造函数

> 答案：**C**。

6. 关于构造函数，下列说法正确的是：
   - A. 返回类型必须写 `void`
   - B. 可以根据参数列表重载
   - C. 不能带参数
   - D. 可以返回当前对象引用

> 答案：**B**。 解析：构造函数没有返回类型。

7. 关于析构函数，下列说法正确的是：
   - A. 可以通过参数列表重载
   - B. 可以返回 `bool`
   - C. 一个类只能有一个析构函数
   - D. 必须显式调用

> 答案：**C**。 解析：析构函数无参数，不能重载。

8. 哪类函数没有当前对象的 `this` 指针？
   - A. 普通非静态成员函数
   - B. `const` 成员函数
   - C. 静态成员函数
   - D. 析构函数

> 答案：**C**。 解析：友元函数也没有，但它不在选项的成员函数类别中。

9. 传统写法中，类内声明 `static int count;` 后，正确的类外定义是：
    - A. `static int A::count=0;`
    - B. `int A::count=0;`
    - C. `int count=0;`
    - D. `A::static int count=0;`

> 答案：**B**。 解析：类外定义不再写 `static`。

10. 静态成员函数可以直接访问的是：
    - A. 任意对象的私有数据成员
    - B. 当前对象的非静态成员
    - C. 类的静态成员
    - D. `this` 指向的成员

> 答案：**C**。

11. 空的默认构造函数对基础类型数据成员的效果通常是：
    - A. 全部置 0
    - B. 全部置 `nullptr`
    - C. 未显式初始化的成员可能是不确定值
    - D. 自动调用 `memset`

> 答案：**C**。

## 二、判断题

1. C++ 的 `struct` 不能包含成员函数。

> 答案：**错**。 解析：C++ 的 `struct` 也能有成员函数、构造函数等。

2. 类定义的右花括号后需要分号。

> 答案：**对**。

3. 仅返回类型不同可以构成函数重载。

> 答案：**错**。

4. `const` 对象可以调用任意非 `const` 成员函数。

> 答案：**错**。

5. 友元函数不是类成员，因此没有该类的 `this` 指针。

> 答案：**对**。

6. 静态数据成员由类的所有对象共享。

> 答案：**对**。
