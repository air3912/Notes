# From C to C++：选择与判断题

> 覆盖第1、2周。单选题均只有一个正确答案。

## 一、单选题

1. 不依赖任何 using 指令，要用完全限定名访问标准输出对象，应写：
   - A. `::cout`
   - B. `std::cout`
   - C. `using::cout`
   - D. `namespace::cout`

> 答案：**B**。 解析：标准输出对象 `cout` 声明在 `std` 命名空间中，其完全限定名是 `std::cout`。

2. 关于引用，下列说法正确的是：
   - A. 引用可以先声明，以后再初始化
   - B. 引用绑定后可以改为引用另一个对象
   - C. 给引用赋值会修改其所绑定的对象
   - D. 普通引用可以取 `nullptr`

> 答案：**C**。 解析：引用必须初始化且不能改绑。

3. 下列函数返回值中会形成悬垂引用的是：

```cpp
int& f() {
    int x = 1;
    return x;
}
```

   - A. 因为 `x` 是 `const`
   - B. 因为 `x` 在函数返回后生命周期结束
   - C. 因为引用不能作为返回类型
   - D. 因为返回值必须是指针

> 答案：**B**。 解析：局部自动变量在函数返回时销毁。

4. `std::string::length()` 的返回类型是：
   - A. `int`
   - B. `unsigned int`
   - C. `size_t`
   - D. `long`

> 答案：**C**。 解析：不是 `int`。

5. 关于 `string::compare` 的返回值，标准保证的是：
   - A. 只可能返回 `-1`、`0`、`1`
   - B. 只可能返回 `true` 或 `false`
   - C. 分别为小于 0、等于 0、大于 0
   - D. 返回第一个不同字符的下标

> 答案：**C**。 解析：非零结果不保证恰好为 `-1` 或 `1`。

6. 下列哪种差异不能单独构成函数重载？
   - A. 参数个数不同
   - B. 参数类型不同
   - C. 参数顺序不同
   - D. 返回类型不同

> 答案：**D**。 解析：函数签名不能仅靠返回类型区分。

7. 下列声明中，`x` 不一定是编译期常量的是：
   - A. `constexpr int x = 5;`
   - B. 函数形参 `const int x`
   - C. 枚举常量 `X`
   - D. 整数字面量 `5`

> 答案：**B**。 解析：运行时传入的 `const` 形参只是只读变量。

8. 下列 `auto` 声明合法的是：
   - A. `auto x;`
   - B. `auto x=1, y=2.0;`
   - C. `auto x=1, y=2;`
   - D. `auto(3) x;`

> 答案：**C**。 解析：`auto` 必须初始化，同一条声明必须推导为相同类型。

9. 默认流格式下，`cout << true;` 通常输出：
   - A. `true`
   - B. `1`
   - C. `TRUE`
   - D. 由 `sizeof(bool)` 决定

> 答案：**B**。 解析：未启用 `boolalpha` 时输出 1/0。

## 二、判断题

1. `endl` 只负责输出换行符，不刷新输出流。

> 答案：**错**。 解析：`endl` 还会刷新输出流。

2. 多个同名命名空间块会组成同一个命名空间。

> 答案：**对**。

3. C++ 可以定义“引用数组”。

> 答案：**错**。 解析：引用不是对象，不能构成引用数组。

4. `c_str()` 返回的类型是 `const char*`。

> 答案：**对**。

5. `const` 在所有场景下都表示编译期常量。

> 答案：**错**。 解析：`const` 对象不一定是常量表达式。

6. `sizeof(bool)` 必然等于 1。

> 答案：**错**。 解析：其大小由实现决定。
