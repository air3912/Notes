1. 循环类型
```cpp
double calcTotalCost(const vector<Transport*>& transports, int weight){
    double ans=0;
    for(auto it=transports.begin();it!=transports.end();it++){//it是指针的迭代器
        ans+=(*it)->cost(weight);
    }
    return ans;
}//这里内部循环的东西是transport*，循环使用的也是迭代器，所以需要解引用一次再作为指针使用。
```
