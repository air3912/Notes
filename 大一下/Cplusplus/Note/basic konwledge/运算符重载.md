### 关于引用——总结就是看怎么传参和怎么返回
- 我先个人总结一下，我觉得需要注意的就是参数，一个是拷贝构造，一个是=运算符的重载，前者因为递归必须传引用，后者则因为需要进行必要的地址检查（this!=&other）也必须为引用
- 反观返回值是否使用引用，我觉得看这个函数返回的东西是否可以作为左值就行，还是只能作为右值即可
- 至于流式操作符，流式输出输入这东西是稳定唯一的
- 最后再对自增前置后置有个印象就完事了这节。


首先是普通加减乘除，直接返回值即可，如果返回引用{
    如果返回临时变量的引用，那直接炸了
    else if返回前者的引用，那么对于有些情况，比如c=a+b:会额外更改a的值，懂你意思
}

然后是后置自增运算符，我们需要返回class(x++),如果返回引用，就是返回一个內部局部变量的引用


然后是一定要用引用的场合：
首先就是拷贝构造，这里有个很符合逻辑的东西，传参数时一定要传参数的引用
```cpp
Integer(const Integer& other) { 
    x = other.x; 
}//没有返回值，但是传入参数必须是引用，如果不是，传入过程中就需要进行一次拷贝，但是由于拷贝构造里还需要拷贝，拷贝就会无线递归，无法进行下去
```
然后是赋值运算符的重载
```cpp
Integer& operator = (const Integer& other) {
    x = other.x;
    return *this; 
}
//可以看到我们就算不return也达到了赋值的效果，但是一样要返回这个东西，为了方便链式赋值a=b=c;这种
//还有一点就是返回值如果不用引用会导致返回的值无法作为左值，再次进行赋值会产生错误（a=b)=c;会与原本含义不符
```

1. 如果是成员函数：

> =/[]/()/->这几个是必须作为成员函数重载的

- +=/-=/*=/这几个返回引用
```cpp
    Vector2D& operator+=(const Vector2D& rhs) {
        this->x += rhs.x;
        this->y += rhs.y;
        return *this; // 返回自身的引用，支持链式调用，如 (a += b) += c;
    }
```
- 成员内+-*/就直接返回数值
```cpp
    Vector2D operator+(const Vector2D& rhs) const {
        return Vector2D(this->x + rhs.x, this->y + rhs.y);
    }//防止b=a+c这种情况
```
- 自增/减
```cpp
    Counter& operator++() {
        ++(this->value);
        return *this; 
    }//前置，不需要int占位符，然后++a依旧可以作为左值赋值所以有用

    Counter operator++(int) {
        Counter temp = *this; // 1. 备份当前状态
        ++(*this);            // 2. 直接调用刚才写好的前置 ++ 完成自增
        return temp;          // 3. 返回备份的旧状态（注意：不能返引用，必须是拷贝）
    }//后置返回数值足矣
```
- 赋值运算
```cpp
    MyIntArray& operator=(const MyIntArray& rhs) {
        if (this == &rhs) {                          //1.如果指向同一位置，下一步delete资源释放没了
            return *this; 
        }
        delete[] this->data;
        this->size = rhs.size;
        if (this->size > 0) {
            this->data = new int[this->size];
            for (size_t i = 0; i < this->size; ++i) {
                this->data[i] = rhs.data[i];
            }
        } else {
            this->data = nullptr;
        }
        return *this;
    }//返回引用，连续赋值，然后很重要的一点是传参数，如果没有那种拷贝构造函数的话，一定要是引用传参喔。
```
- 下标运算
```cpp
    int& operator[](size_t index) {
        return data[index]; 
    }

    const int& operator[](size_t index) const {
        return data[index];
    }//一般要写两个，记得const版本。然后返回引用。
```
