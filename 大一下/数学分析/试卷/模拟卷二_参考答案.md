### 一、 反常积分解答
**【题目】** 讨论 $\int_1^{+\infty} \sin(x^p) \, dx$ ($p > 0$) 的敛散性与绝对/条件收敛性。

**【解析】**
作变量代换：令 $t = x^p \implies x = t^{1/p} \implies dx = \frac{1}{p} t^{1/p - 1} \, dt$。
原积分可写为：
$$I = \frac{1}{p} \int_1^{+\infty} t^{1/p - 1} \sin t \, dt$$

1. **普通收敛性的讨论**：
   在 $[1, +\infty)$ 上，$\int_1^A \sin t \, dt = \cos 1 - \cos A$ 是有界函数。
   设 $g(t) = t^{1/p - 1}$。
   * 若 $1/p - 1 < 0 \implies p > 1$：
     $g(t)$ 单调且 $\lim_{t\to+\infty} g(t) = 0$。根据 Dirichlet 判别法，此时积分收敛。
   * 若 $1/p - 1 \ge 0 \implies 0 < p \le 1$：
     $g(t)$ 不趋于 0（为常数 1 或单调递增趋于 $+\infty$）。在此情况下，被积函数在无穷远处的振幅不趋于 0，积分发散。
   * **结论**：当且仅当 **$p > 1$** 时，原积分收敛。

2. **绝对收敛性的讨论**：
   我们需要判断 $\int_1^{+\infty} |t^{1/p-1} \sin t| \, dt$ 的收敛性。
   因为 $|\sin t| \ge \sin^2 t = \frac{1 - \cos 2t}{2}$，有：
   $$\int_1^{+\infty} t^{1/p-1} |\sin t| \, dt \ge \frac{1}{2} \int_1^{+\infty} t^{1/p-1} \, dt - \frac{1}{2} \int_1^{+\infty} t^{1/p-1} \cos 2t \, dt$$
   * 当 $p > 1$ 时，根据 Dirichlet 判别法，第二项 $\int_1^{+\infty} t^{1/p-1} \cos 2t \, dt$ 总是收敛的。
   * 第一项 $\int_1^{+\infty} t^{1/p-1} \, dt$ 对应 $q$-积分，由于其指数为 $1/p-1 > -1$，显然发散。
   * 因此，当 $p > 1$ 时，积分不绝对收敛。
   * **结论**：原积分在 **$p > 1$** 时为**条件收敛**，且对任意 $p > 0$ **都不绝对收敛**。

### 二、 函数项级数解答
**【题目】** 设 $f_n(x) = n x (1-x^2)^n$ ($x \in [0,1]$)，求极限函数，讨论一致收敛性，并判定积分与极限是否可交换。

**【解析】**
1. **极限函数**：
   * 当 $x = 0$ 时，对所有 $n$ 有 $f_n(0) = 0 \implies \lim_{n\to\infty} f_n(0) = 0$。
   * 当 $x \in (0, 1]$ 时，由于 $0 \le 1-x^2 < 1$，由指数函数与多项式极限关系：
     $$\lim_{n\to\infty} n (1-x^2)^n = 0 \implies \lim_{n\to\infty} f_n(x) = 0$$
   * 综上，极限函数为 **$f(x) = 0 \quad (x \in [0, 1])$**。

2. **一致收敛性**：
   对 $f_n(x)$ 在 $[0, 1]$ 上求导数找极大值：
   $$f_n'(x) = n(1-x^2)^n - 2n^2 x^2 (1-x^2)^{n-1} = n(1-x^2)^{n-1} [ 1 - (2n+1)x^2 ]$$
   唯一驻点为 $x_n = \frac{1}{\sqrt{2n+1}} \in [0, 1]$。其极大值为：
   $$M_n = \sup_{x\in[0,1]} |f_n(x) - f(x)| = f_n(x_n) = \frac{n}{\sqrt{2n+1}} \left( 1 - \frac{1}{2n+1} \right)^n$$
   当 $n \to \infty$ 时，利用重要极限有 $\left( 1 - \frac{1}{2n+1} \right)^n \to e^{-1/2}$。
   而第一部分 $\frac{n}{\sqrt{2n+1}} \sim \frac{\sqrt{n}}{\sqrt{2}} \to +\infty$。
   所以 $\lim_{n\to\infty} M_n = +\infty \ne 0$。
   故 $f_n(x)$ 在区间 $[0, 1]$ 上**非一致收敛**。

3. **极限与积分的交换**：
   * 极限的积分：$\int_0^1 f(x) \, dx = \int_0^1 0 \, dx = 0$。
   * 积分的极限：
     $$\int_0^1 f_n(x) \, dx = \int_0^1 n x (1-x^2)^n \, dx = n \left[ -\frac{(1-x^2)^{n+1}}{2(n+1)} \right]_0^1 = \frac{n}{2(n+1)}$$
     $$\lim_{n\to\infty} \int_0^1 f_n(x) \, dx = \lim_{n\to\infty} \frac{n}{2(n+1)} = \frac{1}{2}$$
   * 由于 $\frac{1}{2} \ne 0$，所以 **$\lim_{n\to\infty} \int_0^1 f_n(x) \, dx = \int_0^1 f(x) \, dx$ 不成立**。
   原因在于该函数序列在闭区间 $[0, 1]$ 上不满足一致收敛性。

### 三、 幂级数解答
**【题目】** 将 $f(x) = \ln(x + \sqrt{1+x^2})$ 展开为关于 $x$ 的幂级数并确定收敛域。

**【解析】**
1. **求导数**：
   $$f'(x) = \frac{1 + \frac{x}{\sqrt{1+x^2}}}{x + \sqrt{1+x^2}} = \frac{1}{\sqrt{1+x^2}} = (1+x^2)^{-1/2}$$
2. **利用广义二项式定理展开**：
   由二项式展开式 $(1+t)^\alpha = 1 + \alpha t + \frac{\alpha(\alpha-1)}{2!} t^2 + \dots$：
   当 $|x| < 1$ 时：
   $$(1+x^2)^{-1/2} = 1 + \sum_{n=1}^{\infty} (-1)^n \frac{(2n-1)!!}{(2n)!!} x^{2n}$$
3. **逐项积分求原函数**：
   因为 $f(0) = 0$，将上式从 0 到 $x$ 逐项积分：
   $$f(x) = \int_0 x (1+t^2)^{-1/2} \, dt = x + \sum_{n=1}^{\infty} (-1)^n \frac{(2n-1)!!}{(2n)!! (2n+1)} x^{2n+1}$$
4. **收敛域确定**：
   * 收敛半径 $R = 1$。
   * 当 $x = 1$ 时，级数为 $1 + \sum_{n=1}^{\infty} (-1)^n \frac{(2n-1)!!}{(2n)!! (2n+1)}$。
     其绝对值项 $u_n = \frac{(2n-1)!!}{(2n)!! (2n+1)}$ 单调递减趋于 0，根据 Leibniz 判别法，级数收敛。
   * 当 $x = -1$ 时，由奇函数对称性，该级数也收敛。
   * **收敛域为 $[-1, 1]$**。

### 四、 傅里叶级数解答
**【题目】** 将 $f(x) = x$ 在 $(0, \pi)$ 上展开为余弦与正弦级数，并求 $\sum_{n=1}^{\infty} \frac{1}{(2n-1)^2}$。

**【解析】**
1. **展开为傅里叶余弦级数（偶延拓）**：
   将 $f(x)$ 偶延拓至 $[-\pi, \pi]$，有 $b_n = 0$。
   $$a_0 = \frac{2}{\pi} \int_0^{\pi} x \, dx = \pi$$
   $$a_n = \frac{2}{\pi} \int_0^{\pi} x \cos nx \, dx = \frac{2}{\pi} \left[ x \frac{\sin nx}{n} \right]_0^{\pi} - \frac{2}{n\pi} \int_0^{\pi} \sin nx \, dx = \frac{2}{\pi n^2} (\cos n\pi - 1) = \frac{2( (-1)^n - 1 )}{\pi n^2}$$
   当 $n$ 为偶数时，$a_n = 0$；当 $n$ 为奇数 $2k-1$ 时，$a_{2k-1} = -\frac{4}{\pi(2k-1)^2}$。
   余弦级数展开式为：
   $$x = \frac{\pi}{2} - \frac{4}{\pi} \sum_{k=1}^{\infty} \frac{\cos(2k-1)x}{(2k-1)^2} \quad (x \in [0, \pi])$$

2. **展开为傅里叶正弦级数（奇延拓）**：
   将 $f(x)$ 奇延拓至 $[-\pi, \pi]$，有 $a_n = 0$。
   $$b_n = \frac{2}{\pi} \int_0^{\pi} x \sin nx \, dx = \frac{2}{\pi} \left[ -x \frac{\cos nx}{n} \right]_0^{\pi} + \frac{2}{n\pi} \int_0^{\pi} \cos nx \, dx = -\frac{2}{n} (-1)^n = \frac{2(-1)^{n+1}}{n}$$
   正弦级数展开式为：
   $$x = 2 \sum_{n=1}^{\infty} \frac{(-1)^{n+1}}{n} \sin nx \quad (x \in (0, \pi))$$

3. **求数项级数和**：
   在余弦级数展开式中取 $x = 0$，由于偶延拓函数在 $x=0$ 处连续：
   $$0 = \frac{\pi}{2} - \frac{4}{\pi} \sum_{k=1}^{\infty} \frac{1}{(2k-1)^2} \implies \sum_{k=1}^{\infty} \frac{1}{(2k-1)^2} = \frac{\pi^2}{8}$$

### 五、 多元极限与连续性解答
**【题目】** 证明极限不存在；并讨论 $f(x,y) = \sqrt{|xy|}$ 在 $(0,0)$ 处的连续与可微性。

**【解析】**
1. **极限不存在证明**：
   * 路径 1：沿直线 $y = x$ 趋近于 $(0,0)$。
     $$\lim_{x\to 0} \frac{x^4}{x^4 + 0} = 1$$
   * 路径 2：沿坐标轴 $y = 0$ 趋近于 $(0,0)$。
     $$\lim_{x\to 0} \frac{0}{0 + x^2} = 0$$
   由于从不同路径得到的极限值不相等，原极限不存在。

2. **$f(x,y) = \sqrt{|xy|}$ 连续与可微性**：
   * **连续性**：
     $$|f(x,y) - f(0,0)| = \sqrt{|xy|} \le \sqrt{\frac{x^2+y^2}{2}}$$
     当 $(x,y) \to (0,0)$ 时，右边显然趋于 0。故在 $(0,0)$ 处连续。
   * **偏导数**：
     $$f_x(0,0) = \lim_{h\to 0} \frac{f(h,0) - f(0,0)}{h} = \lim_{h\to 0} \frac{0 - 0}{h} = 0$$
     $$f_y(0,0) = \lim_{k\to 0} \frac{f(0,k) - f(0,0)}{k} = \lim_{k\to 0} \frac{0 - 0}{k} = 0$$
   * **可微性**：
     若可微，则有：
     $$\lim_{(h,k)\to(0,0)} \frac{f(h,k) - f(0,0) - f_x(0,0)h - f_y(0,0)k}{\sqrt{h^2+k^2}} = \lim_{(h,k)\to(0,0)} \frac{\sqrt{|hk|}}{\sqrt{h^2+k^2}} = 0$$
     设沿路径 $k = h > 0$ 趋于 0：
     $$\lim_{h\to 0^+} \frac{\sqrt{h^2}}{\sqrt{2h^2}} = \frac{1}{\sqrt{2}} \ne 0$$
     所以该极限不为 0，即函数在 $(0,0)$ 处不可微。

### 六、 偏导数与全微分解答
**【题目】** $e^z + xz + y = 0$ 确定 $z(x,y)$，求在点 $(0, -1)$ 处的二阶泰勒多项式。

**【解析】**
已知在 $(0, -1)$ 处有 $z = 0$。
方程两边分别对 $x, y$ 求一阶偏导数：
* 对 $x$ 求导：$e^z z_x + z + x z_x = 0$。将点 $(0, -1, 0)$ 代入，得 $1 \cdot z_x + 0 + 0 = 0 \implies z_x(0, -1) = 0$。
* 对 $y$ 求导：$e^z z_y + x z_y + 1 = 0$。将点 $(0, -1, 0)$ 代入，得 $1 \cdot z_y + 0 + 1 = 0 \implies z_y(0, -1) = -1$。

对方程再次求导数：
* 对 $x$ 求导方程再次对 $x$ 求导：$e^z z_x^2 + e^z z_{xx} + 2z_x + x z_{xx} = 0$。
  将点 $(0, -1, 0)$ 及一阶导数代入，得 $1 \cdot (0) + 1 \cdot z_{xx} + 0 + 0 = 0 \implies z_{xx}(0,-1) = 0$。
* 对 $x$ 求导方程对 $y$ 求导：$e^z z_y z_x + e^z z_{xy} + z_y + z_y + x z_{xy} = 0$。
  代入点 $(0, -1, 0)$ 及一阶导数，得 $0 + 1 \cdot z_{xy} - 1 - 1 + 0 = 0 \implies z_{xy}(0,-1) = 2$。
* 对 $y$ 求导方程对 $y$ 求导：$e^z z_y^2 + e^z z_{yy} + x z_{yy} = 0$。
  代入点 $(0, -1, 0)$ 及一阶导数，得 $1 \cdot (-1)^2 + 1 \cdot z_{yy} + 0 = 0 \implies z_{yy}(0,-1) = -1$。

泰勒多项式（自变量增量为 $x-0 = x$ 与 $y - (-1) = y+1$）：
$$T_2(x, y) = z(0, -1) + z_x x + z_y (y+1) + \frac{1}{2} \left[ z_{xx} x^2 + 2 z_{xy} x(y+1) + z_{yy} (y+1)^2 \right]$$
$$T_2(x, y) = -(y+1) + 2x(y+1) - \frac{1}{2}(y+1)^2$$
展开整理为：
$$T_2(x, y) = -y - 1 + 2xy + 2x - \frac{1}{2}(y+1)^2$$

### 七、 极值与条件极值解答
**【题目】** 求曲线 $\begin{cases} x^2 + y^2 = 1 \\ x + y + z = 1 \end{cases}$ 到原点的最大和最小距离。

**【解析】**
目标函数为距离平方 $f(x,y,z) = x^2+y^2+z^2$。约束条件为 $g_1 = x^2+y^2-1=0$ 且 $g_2 = x+y+z-1=0$。
构造拉格朗日函数：
$$L(x,y,z,\lambda,\mu) = x^2+y^2+z^2 - \lambda(x^2+y^2-1) - \mu(x+y+z-1)$$
求偏导数：
1) $L_x = 2x - 2\lambda x - \mu = 0$
2) $L_y = 2y - 2\lambda y - \mu = 0$
3) $L_z = 2z - \mu = 0 \implies \mu = 2z$
4) 约束条件：$x^2+y^2=1$, $x+y+z=1$

由 1)、2) 得：$2x(1-\lambda) = \mu$ 且 $2y(1-\lambda) = \mu$。
* **情况 1**：若 $\lambda = 1$，则必有 $\mu = 0 \implies z = 0$。
  将 $z=0$ 代入第二个约束条件得 $x+y=1$。
  联立 $x^2+y^2=1$ 得驻点：$(1, 0, 0)$ 和 $(0, 1, 0)$。
  此时对应的距离平方为 $f = 1$。
* **情况 2**：若 $\lambda \ne 1$，则 $x = y$。
  由第一个约束条件 $2x^2 = 1 \implies x = y = \pm \frac{1}{\sqrt{2}}$。
  * 当 $x = y = \frac{1}{\sqrt{2}}$ 时，由第二个约束得 $z = 1 - \sqrt{2}$。
    此点距离平方 $f = 1 + (1-\sqrt{2})^2 = 4 - 2\sqrt{2} \approx 1.17$。
  * 当 $x = y = -\frac{1}{\sqrt{2}}$ 时，得 $z = 1 + \sqrt{2}$。
    此点距离平方 $f = 1 + (1+\sqrt{2})^2 = 4 + 2\sqrt{2} \approx 6.83$。

**比较结论**：
* 最小距离为 $d_{\min} = \sqrt{1} = 1$（在点 $(1,0,0)$ 或 $(0,1,0)$ 处取得）；
* 最大距离为 $d_{\max} = \sqrt{4+2\sqrt{2}}$（在点 $(-\frac{1}{\sqrt{2}}, -\frac{1}{\sqrt{2}}, 1+\sqrt{2})$ 处取得）。

### 八、 含参变量的正常积分解答
**【题目】** 计算 $\int_0^1 \frac{(x-1)^2}{\ln x} \, dx$。

**【解析】**
已知 $\int_0^1 \frac{x^k-1}{\ln x} \, dx = \ln(k+1)$（为方便，令该动积分为 $J(k)$，其中 $k > -1$）。
将分子展开：
$$(x-1)^2 = x^2 - 2x + 1 = (x^2 - 1) - 2(x - 1)$$
代回被积式，根据积分的线性性质：
$$\int_0^1 \frac{(x-1)^2}{\ln x} \, dx = \int_0^1 \frac{x^2 - 1}{\ln x} \, dx - 2 \int_0^1 \frac{x - 1}{\ln x} \, dx = J(2) - 2J(1)$$
将已知公式 $J(k) = \ln(k+1)$ 分别代入：
* $J(2) = \ln 3$
* $J(1) = \ln 2$
所以：
$$\int_0^1 \frac{(x-1)^2}{\ln x} \, dx = \ln 3 - 2\ln 2 = \ln\frac{3}{4}$$
原积分值为 **$\ln\frac{3}{4}$**。

### 九、 重积分解答
**【题目】** 计算 $\iiint_{\Omega} z \, dV$。

**【解析】**
引入球面坐标 $x = r\sin\phi\cos\theta, y = r\sin\phi\sin\theta, z = r\cos\phi$，体积微元 $dV = r^2\sin\phi \, dr d\phi d\theta$。
确定积分区域 $\Omega$ 的球面参数范围：
* 球面约束为 $r \le R$。
* 锥面约束为 $z \ge \sqrt{x^2+y^2} \implies r\cos\phi \ge r\sin\phi \implies \tan\phi \le 1 \implies 0 \le \phi \le \frac{\pi}{4}$。
* 根据题目位于第一、二卦限部分（在 $xy$ 平面上对应 $y \ge 0$ 的上半圆），所以极角 $\theta$ 的范围为 $0 \le \theta \le \pi$。
进行累次积分：
$$I = \int_0^{\pi} d\theta \int_0^{\pi/4} d\phi \int_0^R (r\cos\phi) r^2\sin\phi \, dr$$
$$I = \left( \int_0^{\pi} d\theta \right) \cdot \left( \int_0^{\pi/4} \sin\phi\cos\phi \, d\phi \right) \cdot \left( \int_0^R r^3 \, dr \right)$$
分别计算三个部分积分值：
1. $\int_0^{\pi} d\theta = \pi$
2. $\int_0^{\pi/4} \sin\phi\cos\phi \, d\phi = \left[ \frac{1}{2}\sin^2\phi \right]_0^{\pi/4} = \frac{1}{4}$
3. $\int_0^R r^3 \, dr = \frac{R^4}{4}$
相乘得到结果：
$$I = \pi \cdot \frac{1}{4} \cdot \frac{R^4}{4} = \frac{\pi R^4}{16}$$

### 十、 曲线与曲面积分解答
**【题目】** 计算 $\iint_{\Sigma} x^3 \, dydz + y^3 \, dzdx + z^3 \, dxdy$，球面外侧。

**【解析】**
设 $\Sigma$ 围成的闭区域为球体 $V: x^2+y^2+z^2 \le R^2$。由于 $\Sigma$ 取外侧，可直接应用 **Gauss 散度公式**。
向量场为 $\vec{F} = (x^3, y^3, z^3)$，其散度为：
$$\text{div} \vec{F} = \frac{\partial(x^3)}{\partial x} + \frac{\partial(y^3)}{\partial y} + \frac{\partial(z^3)}{\partial z} = 3x^2 + 3y^2 + 3z^2 = 3(x^2+y^2+z^2)$$
应用高斯散度公式得：
$$I = \iiint_V 3(x^2+y^2+z^2) \, dV$$
利用球面坐标代入计算：
$$I = 3 \int_0^{2\pi} d\theta \int_0^\pi d\phi \int_0^R r^2 \cdot r^2\sin\phi \, dr$$
$$I = 3 \left( \int_0^{2\pi} d\theta \right) \cdot \left( \int_0^\pi \sin\phi \, d\phi \right) \cdot \left( \int_0^R r^4 \, dr \right)$$
分别计算各项积分：
* $\int_0^{2\pi} d\theta = 2\pi$
* $\int_0^\pi \sin\phi \, d\phi = 2$
* $\int_0^R r^4 \, dr = \frac{R^5}{5}$
代入整理：
$$I = 3 \cdot 2\pi \cdot 2 \cdot \frac{R^5}{5} = \frac{12\pi R^5}{5}$$
故积分值为 **$\frac{12\pi R^5}{5}$**。
