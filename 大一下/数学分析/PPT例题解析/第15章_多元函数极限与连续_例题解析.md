# 工科数学分析期末大题突破：第 15 章 多元函数的极限与连续性（PPT例题与详解）

> **期末核心导读**：二元函数的极限性质是后续微积分计算的理论根基。在期末大题中，最常考的题型包括：**二重极限存在性的判定（通过极坐标或夹逼定理求极限）**，以及**证明二重极限不存在（通过直线路径 $y=kx$ 或抛物线路径 $x=ky^2$ 等寻找不同极限值）**。此外，**讨论分段函数在分界点处的连续性**也是常见的简答/计算题。

---

## 一、 考情分析与考查频率

| 大题考点 | 常见考查形式 | 考查频率 |
| :--- | :--- | :---: |
| **证明二重极限不存在** | 构造多条路径（如 $y=kx$ 或 $x=ky^2$）使得极限值不同，否定二重极限的存在（期末大题最爱） | ★★★★★ |
| **计算二重极限** | 利用极坐标代换（$x=r\cos\theta, y=r\sin\theta$）或夹逼定理求极限值 | ★★★★☆ |
| **二重极限与累次极限关系** | 区分全面极限与两个累次极限，考查三者同存在但不等、或部分存在的性质 | ★★★☆☆ |
| **分段函数连续性讨论** | 讨论 $f(x,y)$ 在 $(0,0)$ 处的极限与定义值的关系，判断是否连续或进行补定义 | ★★★☆☆ |

## 二、 核心公式与概念（必背）

### 1. 二重极限（全面极限）
二重极限 $\lim_{(x,y) \to (x_0, y_0)} f(x,y) = A$ 指的是：自变量点 $(x,y)$ 以**任意路径、任意方式**无限逼近 $(x_0, y_0)$ 时，函数值都无限接近于 $A$。
* **极坐标判定法（最常用）**：若在 $(0,0)$ 附近令 $x = r\cos\theta, y = r\sin\theta$，且被积式化为：
  $$f(x,y) - A = g(r, \theta)$$
  如果存在与 $\theta$ 无关的非负函数 $h(r)$，使得 $|g(r, \theta)| \le h(r)$ 且 $\lim_{r\to 0^+} h(r) = 0$，则二重极限存在且为 $A$。

### 2. 累次极限
先对一个自变量求极限，再对另一个自变量求极限：
$$L_{12} = \lim_{x\to x_0} \left( \lim_{y\to y_0} f(x,y) \right), \quad L_{21} = \lim_{y\to y_0} \left( \lim_{x\to x_0} f(x,y) \right)$$
* **极重要理论结论**：
  1. **若 $L_{12} \ne L_{21}$，则二重极限一定不存在**。
  2. 即使 $L_{12} = L_{21}$，二重极限也不一定存在。
  3. 二重极限存在，累次极限不一定存在（如 $f(x,y) = x \sin(1/y)$ 处）。

## 三、 PPT例题详解：§1 平面点集

### 例 2.1（闭包与边界的判定）
**【题目】** 在 $\mathbb{R}^2$ 上，设 $S = \{(x,y) \mid 1 < x^2+y^2 < 4\}$，写出其内点、边界点、聚点和闭包。
* **考点**：平面点集的拓扑概念。
* **解析**：
  1. **内点**：由于 $S$ 是由严格不等式定义的开圆环，其内每一点都是内点，故内点集 $\text{int}(S) = S$。
  2. **边界点**：边界为两个圆周，即 $\partial S = \{(x,y) \mid x^2+y^2 = 1 \text{ 或 } x^2+y^2 = 4\}$。
  3. **聚点**：圆环内的所有点以及两个边界圆周上的所有点均是聚点。
  4. **闭包**：闭包是原集合与其边界的并集：
     $$\bar{S} = \{(x,y) \mid 1 \le x^2+y^2 \le 4\}$$

## 四、 PPT例题详解：§2 多元函数极限与连续

### 例 2.1（夹逼估计求二重极限）
**【题目】** 证明：$\lim_{(x,y) \to (0,0)} (x+y) \sin \frac{1}{x^2+y^2} = 0$。
* **考点**：有界函数与无穷小的乘积。
* **解析**：
  由于三角函数有界：对一切非零点，有 $\left| \sin \frac{1}{x^2+y^2} \right| \le 1$。
  利用绝对值不等式：
  $$0 \le \left| (x+y) \sin \frac{1}{x^2+y^2} - 0 \right| \le |x+y| \le |x| + |y|$$
  因为 $\lim_{(x,y)\to(0,0)} (|x| + |y|) = 0$，由夹逼定理有：
  $$\lim_{(x,y) \to (0,0)} (x+y) \sin \frac{1}{x^2+y^2} = 0$$

---

### 例 2.2（直线路径否定极限存在）
**【题目】** 证明：$f(x,y) = \frac{xy}{x^2+y^2}$ 在 $(0,0)$ 处的二重极限不存在。
* **考点**：用直线路径判定极限不存在。
* **解析**：
  若我们让点 $(x,y)$ 沿直线 $y = kx$ 趋于 $(0,0)$。
  代入函数式：
  $$f(x, kx) = \frac{x(kx)}{x^2 + (kx)^2} = \frac{k x^2}{x^2 (1+k^2)} = \frac{k}{1+k^2} \quad (x \ne 0)$$
  求极限得：
  $$\lim_{x\to 0} f(x, kx) = \frac{k}{1+k^2}$$
  由于此极限值依赖于斜率 $k$ 的选择：
  * 若沿 $y=x$（$k=1$）逼近，极限为 $\frac{1}{2}$；
  * 若沿 $y=0$（$k=0$，即 $x$ 轴）逼近，极限为 $0$。
  由于从不同方向逼近得到的极限不同，故原二重极限不存在。

---

### 例 2.3（直线全部相同但曲线不同的经典反例）
**【题目】** 讨论 $f(x,y) = \frac{xy^2}{x^2+y^4}$ 在 $(0,0)$ 处的二重极限是否存在。
* **考点**：直线路径均相同，但抛物线路径不同的反例（期末压轴高频大题）。
* **解析**：
  1. **沿任意直线路径 $y = kx$ 逼近**：
     $$f(x, kx) = \frac{x(kx)^2}{x^2 + (kx)^4} = \frac{k^2 x^3}{x^2 (1 + k^4 x^2)} = \frac{k^2 x}{1 + k^4 x^2} \quad (x \ne 0)$$
     求极限：
     $$\lim_{x\to 0} f(x, kx) = \lim_{x\to 0} \frac{k^2 \cdot 0}{1 + 0} = 0$$
     当沿 $x=0$（$y$ 轴）逼近时，显然 $f(0, y) = 0 \to 0$。
     这意味着，**自变量沿任意一条经过原点的直线逼近时，极限值全部为 0**。
  2. **沿抛物线路径 $x = y^2$ 逼近**：
     代入函数式：
     $$f(y^2, y) = \frac{y^2 \cdot y^2}{(y^2)^2 + y^4} = \frac{y^4}{2y^4} = \frac{1}{2} \quad (y \ne 0)$$
     求极限：
     $$\lim_{y\to 0} f(y^2, y) = \frac{1}{2}$$
  3. **判定**：
     因为 $0 \ne \frac{1}{2}$，通过不同曲线路径得到的极限不同，故原二重极限不存在。
  * **避错提示**：做大题时不能只用 $y=kx$ 判定，若分子分母次数不齐次（如分母有 $x^2$ 和 $y^4$），必须尝试 $x = ky^2$ 路径。

### 例 3.1（二重极限存在，但累次极限不存在的反例）
**【题目】** 设 $f(x,y) = (x^2+y^2)\sin\frac{1}{x^2+y^2} - \cos\frac{1}{x^2+y^2}$ 形式？
不，我们采用更经典的易懂反例：
设 $f(x,y) = x \sin \frac{1}{y} + y \sin \frac{1}{x}$ （$x \ne 0, y \ne 0$），讨论其在 $(0,0)$ 处的二重极限与累次极限。
* **考点**：二重极限与累次极限的区别。
* **解析**：
  1. **二重极限**：
     由于 $|f(x,y)| \le |x| \cdot 1 + |y| \cdot 1 = |x| + |y|$。
     当 $(x,y) \to (0,0)$ 时，$|x| + |y| \to 0$，由夹逼定理二重极限存在且为 0。
  2. **累次极限**：
     计算 $\lim_{x\to 0} \lim_{y\to 0} f(x,y)$。
     内层极限为 $\lim_{y\to 0} (x \sin\frac{1}{y} + y \sin\frac{1}{x})$。
     因为当 $y\to 0$ 时，$\sin\frac{1}{y}$ 在 $[-1, 1]$ 之间无限振荡，对于固定的 $x \ne 0$，该极限不存在。
     所以累次极限 $\lim_{x\to 0} \lim_{y\to 0} f(x,y)$ 不存在。
  * **结论**：二重极限存在，但累次极限可以不存在。

---

### 例 3.3（二重极限不存在，但累次极限存在且相等的反例）
**【题目】** 设 $f(x,y) = \frac{x^2 - y^2}{x^2 + y^2}$，求其在 $(0,0)$ 处的两个累次极限与二重极限。
* **考点**：累次极限与二重极限的独立性。
* **解析**：
  1. **计算累次极限**：
     $$\lim_{x\to 0} \lim_{y\to 0} \frac{x^2 - y^2}{x^2 + y^2} = \lim_{x\to 0} \frac{x^2}{x^2} = 1$$
     $$\lim_{y\to 0} \lim_{x\to 0} \frac{x^2 - y^2}{x^2 + y^2} = \lim_{y\to 0} \frac{-y^2}{y^2} = -1$$
     两个累次极限均存在，但由于 $1 \ne -1$，直接说明二重极限不存在。
  * **补充例题**：若 $f(x,y) = \frac{x^2y^2}{x^4+y^4-x^2y^2}$，可以证明其两个累次极限都是 0，但沿 $y=x$ 二重极限为 1，同样不存在。

### 例 4.5（寻找不连续点集合）
**【题目】** 求函数 $u = \tan(x^2+y^2)$ 的所有不连续点组成的集合。
* **考点**：不连续点集合的求解。
* **解析**：
  由于 $u(t) = \tan t$ 在 $t = \frac{\pi}{2} + k\pi$（$k \in \mathbb{Z}$）处无定义且不连续。
  代入内层函数 $t = x^2+y^2 \ge 0$，可得：
  $$x^2+y^2 = \frac{\pi}{2} + k\pi, \quad k = 0, 1, 2, \dots$$
  * **结论**：该函数的不连续点集为以原点为圆心、半径为 $\sqrt{\frac{\pi}{2} + k\pi}$（$k \ge 0$）的一族同心圆环。

## 五、 备考解题套路与易错防范

### 1. 证明二重极限不存在的路径选择技巧
* **同阶项（如 $x^2+y^2$）**：直接设 $y = kx$。
* **不同阶项（如 $x^2+y^4$）**：观察分子分母次数。为了使分母各项在次数上对齐，若分母是 $x^2+y^4$，就必须令 $x = ky^2$；若分母是 $x^4+y^2$，则令 $y = kx^2$。

### 2. 夹逼估计常用不等式
在大题中若要证明极限存在，最安全的是直接用绝对值夹逼：
* $|xy| \le \frac{1}{2}(x^2+y^2)$
* $x^2 \le x^2+y^2 \implies \frac{x^2}{x^2+y^2} \le 1$
* 利用极坐标 $x = r\cos\theta, y = r\sin\theta$，有 $|x| \le r, |y| \le r$。
利用这些放缩，可以极快地将被积式压至 0。
