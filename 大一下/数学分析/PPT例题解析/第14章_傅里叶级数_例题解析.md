# 工科数学分析期末大题突破：第 14 章 傅里叶级数（PPT例题与详解）

> **期末核心导读**：傅里叶级数在期末考试中通常作为一道独立的计算大题出现。考核的核心点非常集中：**给定周期函数的傅里叶级数展开**、**在半区间 $[0, \pi]$ 或 $[0, l]$ 上进行正弦级数（奇延拓）与余弦级数（偶延拓）展开**，以及**利用收敛定理（Dirichlet 定理）确定间断点/端点处的级数收敛值，并求特定数项级数之和**。

---

## 一、 考情分析与考查频率

| 大题考点 | 常见考查形式 | 考查频率 |
| :--- | :--- | :---: |
| **半区间正弦/余弦展开** | 对 $[0, \pi]$ 上的函数进行奇延拓或偶延拓，求对应的 $b_n$ 或 $a_n$，并写出级数式 | ★★★★★ |
| **利用收敛定理求级数和** | 结合收敛定理，代入特殊点（如 $x = 0, \frac{\pi}{2}, \pi$）求数项级数（如 $\sum \frac{1}{n^2}$ 或 $\sum \frac{(-1)^n}{n^2}$）的和 | ★★★★★ |

| **一般周期 $2l$ 函数的展开** | 将区间及三角项缩放为 $\cos \frac{n\pi x}{l}$ 和 $\sin \frac{n\pi x}{l}$，计算一般周期级数 | ★★★☆☆ |

## 二、 核心公式与收敛定理（必背）

### 1. 周期为 $2\pi$ 的傅里叶级数
若 $f(x)$ 周期为 $2\pi$，在区间 $[-\pi, \pi]$ 上的展开式为：
$$f(x) \sim \frac{a_0}{2} + \sum_{n=1}^{\infty} \left( a_n \cos nx + b_n \sin nx \right)$$
其中系数为：
$$a_0 = \frac{1}{\pi} \int_{-\pi}^{\pi} f(x) dx$$
$$a_n = \frac{1}{\pi} \int_{-\pi}^{\pi} f(x) \cos nx dx$$
$$b_n = \frac{1}{\pi} \int_{-\pi}^{\pi} f(x) \sin nx dx$$

### 2. 奇偶函数的简化（对称性分析）
* 若 $f(x)$ 是 **偶函数**：其正弦系数 $b_n = 0$，傅里叶级数化为**余弦级数**：
  $$f(x) \sim \frac{a_0}{2} + \sum_{n=1}^{\infty} a_n \cos nx \quad \left( a_n = \frac{2}{\pi} \int_0^{\pi} f(x) \cos nx dx \right)$$
* 若 $f(x)$ 是 **奇函数**：其余弦系数 $a_0 = a_n = 0$，傅里叶级数化为**正弦级数**：
  $$f(x) \sim \sum_{n=1}^{\infty} b_n \sin nx \quad \left( b_n = \frac{2}{\pi} \int_0^{\pi} f(x) \sin nx dx \right)$$

### 3. 一般周期为 $2l$ 的傅里叶级数
若 $f(x)$ 周期为 $2l$，在区间 $[-l, l]$ 上的展开式角频率变为 $\frac{n\pi x}{l}$：
$$f(x) \sim \frac{a_0}{2} + \sum_{n=1}^{\infty} \left( a_n \cos \frac{n\pi x}{l} + b_n \sin \frac{n\pi x}{l} \right)$$
其中系数为：
$$a_n = \frac{1}{l} \int_{-l}^{l} f(x) \cos \frac{n\pi x}{l} dx, \quad b_n = \frac{1}{l} \int_{-l}^{l} f(x) \sin \frac{n\pi x}{l} dx$$

### 4. 傅里叶收敛定理（Dirichlet 定理）
设 $f(x)$ 是周期为 $2l$ 的周期函数，且在一个周期内分段光滑。则 $f(x)$ 的傅里叶级数在任意点 $x$ 处收敛于：
$$\bar{S}(x) = \frac{f(x+0) + f(x-0)}{2}$$
* 在 **连续点** 处：级数收敛于 $f(x)$；
* 在 **跳跃间断点** 处：级数收敛于左右极限的算术平均值；
* 在 **区间端点** $\pm l$ 处：级数收敛于 $\frac{f(-l+0) + f(l-0)}{2}$。

## 三、 PPT例题详解：§1 三角级数与傅里叶级数

### 例 2.1（三角函数系的正交性证明）
**【题目】** 证明三角函数族 $\{1, \cos x, \sin x, \dots, \cos nx, \sin nx, \dots\}$ 是任意一个长度为 $2\pi$ 区间（通常取 $[-\pi, \pi]$）上的正交函数列。
* **考点**：三角函数系正交性，积化和差积分（理论基础大题）。
* **证明**：
  根据正交性定义，需证该函数族中任意两个不同函数的乘积在区间 $[-\pi, \pi]$ 上的积分为 0。
  1. **常数项与三角项正交**：
     $$\int_{-\pi}^{\pi} 1 \cdot \cos nx dx = \left[ \frac{\sin nx}{n} \right]_{-\pi}^{\pi} = 0 \quad (n \ge 1)$$
     $$\int_{-\pi}^{\pi} 1 \cdot \sin nx dx = \left[ -\frac{\cos nx}{n} \right]_{-\pi}^{\pi} = 0 \quad (n \ge 1)$$
  2. **正弦与余弦正交**：
     被积函数为奇函数，在对称区间上积分显然为 0：
     $$\int_{-\pi}^{\pi} \sin mx \cos nx dx = 0 \quad (\forall m, n)$$
  3. **同类三角项正交（$m \ne n$）**：
     利用积化和差公式：
     $$\int_{-\pi}^{\pi} \cos mx \cos nx dx = \frac{1}{2} \int_{-\pi}^{\pi} [\cos(m+n)x + \cos(m-n)x] dx = 0 \quad (m \ne n)$$
     $$\int_{-\pi}^{\pi} \sin mx \sin nx dx = \frac{1}{2} \int_{-\pi}^{\pi} [\cos(m-n)x - \cos(m+n)x] dx = 0 \quad (m \ne n)$$
  4. **同项自积（平方积分）**：
     当 $m = n$ 时：
     $$\int_{-\pi}^{\pi} \cos^2 nx dx = \int_{-\pi}^{\pi} \frac{1+\cos 2nx}{2} dx = \pi$$
     $$\int_{-\pi}^{\pi} \sin^2 nx dx = \int_{-\pi}^{\pi} \frac{1-\cos 2nx}{2} dx = \pi$$
  综上，该函数族为正交函数列。

### 例 2.2（绝对值偶函数的傅里叶展开）
**【题目】** 求 $f(x) = |x|$ 在 $[-\pi, \pi]$ 上的傅里叶级数，并证明 $\sum_{k=1}^{\infty} \frac{1}{(2k-1)^2} = \frac{\pi^2}{8}$。
* **考点**：偶函数展开与数项级数求和。
* **解析**：
  1. **系数分析**：
     由于 $f(x) = |x|$ 是偶函数，所以正弦系数 $b_n = 0$。
  2. **计算余弦系数**：
     $$a_0 = \frac{2}{\pi} \int_0^{\pi} x dx = \frac{2}{\pi} \left[ \frac{x^2}{2} \right]_0^{\pi} = \pi$$
     $$a_n = \frac{2}{\pi} \int_0^{\pi} x \cos nx dx$$
     使用分部积分法：
     $$a_n = \frac{2}{\pi} \left( \left[ \frac{x \sin nx}{n} \right]_0^{\pi} - \int_0^{\pi} \frac{\sin nx}{n} dx \right) = -\frac{2}{n\pi} \left[ -\frac{\cos nx}{n} \right]_0^{\pi} = \frac{2 ((-1)^n - 1)}{\pi n^2}$$
     * 当 $n$ 为偶数时，$(-1)^n - 1 = 0 \implies a_n = 0$；
     * 当 $n$ 为奇数时，$(-1)^n - 1 = -2 \implies a_n = -\frac{4}{\pi n^2}$。
  3. **写出级数表达式**：
     由于 $f(x) = |x|$ 在整个闭区间上连续且分段光滑，由收敛定理级数处处收敛于 $f(x)$：
     $$|x| = \frac{\pi}{2} - \frac{4}{\pi} \sum_{k=1}^{\infty} \frac{\cos(2k-1)x}{(2k-1)^2} \quad (x \in [-\pi, \pi])$$
  4. **求数项级数之和**：
     取特殊点 $x = 0$（连续点），代入上式：
     $$0 = \frac{\pi}{2} - \frac{4}{\pi} \sum_{k=1}^{\infty} \frac{\cos 0}{(2k-1)^2} \implies \frac{\pi}{2} = \frac{4}{\pi} \sum_{k=1}^{\infty} \frac{1}{(2k-1)^2}$$
     整理可得：
     $$\sum_{k=1}^{\infty} \frac{1}{(2k-1)^2} = \frac{\pi^2}{8}$$

### 例 2.3（非对称区间上的傅里叶展开）
**【题目】** 设 $f(x) = x^2$（$x \in (0, 2\pi)$），周期 $T = 2\pi$，求其傅里叶级数。
* **考点**：非对称区间的傅里叶展开，积分限调整。
* **解析**：
  因为区间是 $(0, 2\pi)$ 而不是对称区间，我们不能直接使用奇偶性判定，必须直接积分。
  1. **计算 $a_0$**：
     $$a_0 = \frac{1}{\pi} \int_0^{2\pi} x^2 dx = \frac{1}{\pi} \left[ \frac{x^3}{3} \right]_0^{2\pi} = \frac{8\pi^2}{3}$$
  2. **计算 $a_n$**：
     $$a_n = \frac{1}{\pi} \int_0^{2\pi} x^2 \cos nx dx$$
     使用两次分部积分：
     $$\int_0^{2\pi} x^2 \cos nx dx = \left[ \frac{x^2 \sin nx}{n} \right]_0^{2\pi} - \frac{2}{n} \int_0^{2\pi} x \sin nx dx = 0 - \frac{2}{n} \left( \left[ -\frac{x \cos nx}{n} \right]_0^{2\pi} + \int_0^{2\pi} \frac{\cos nx}{n} dx \right)$$
     $$= -\frac{2}{n} \left( -\frac{2\pi}{n} \right) = \frac{4\pi}{n^2}$$
     所以：
     $$a_n = \frac{4}{n^2}$$
  3. **计算 $b_n$**：
     $$b_n = \frac{1}{\pi} \int_0^{2\pi} x^2 \sin nx dx$$
     使用两次分部积分：
     $$\int_0^{2\pi} x^2 \sin nx dx = \left[ -\frac{x^2 \cos nx}{n} \right]_0^{2\pi} + \frac{2}{n} \int_0^{2\pi} x \cos nx dx = -\frac{4\pi^2}{n} + \frac{2}{n} \left[ \frac{\sin nx}{n} \right]_0^{2\pi} = -\frac{4\pi^2}{n}$$
     所以：
     $$b_n = -\frac{4\pi}{n}$$
  4. **傅里叶展开式**：
     在 $(0, 2\pi)$ 上级数收敛于 $x^2$：
     $$x^2 \sim \frac{4\pi^2}{3} + 4 \sum_{n=1}^{\infty} \left( \frac{\cos nx}{n^2} - \frac{\pi \sin nx}{n} \right) \quad (x \in (0, 2\pi))$$
     * **端点收敛分析**：在 $x = 0$（或 $2\pi$）处，级数收敛至周期延拓的左右极限平均值：
       $$\bar{S}(0) = \frac{f(0^+) + f(2\pi^-)}{2} = \frac{0 + 4\pi^2}{2} = 2\pi^2$$

### 例 3.1（半区间正弦与余弦级数双重展开）
**【题目】** 将 $f(x) = x$（$x \in [0, \pi]$）分别展开为余弦级数与正弦级数。
* **考点**：正弦展开（奇延拓）与余弦展开（偶延拓）对比。
* **解析**：
  1. **展开为余弦级数（偶延拓）**：
     偶延拓后只计算 $a_0$ 与 $a_n$。
     根据偶延拓计算公式，其系数等于例 2.2 的计算结果：
     $$a_0 = \pi, \quad a_n = \frac{2 ((-1)^n - 1)}{\pi n^2}$$
     因此余弦级数展开为：
     $$x = \frac{\pi}{2} - \frac{4}{\pi} \sum_{k=1}^{\infty} \frac{\cos(2k-1)x}{(2k-1)^2} \quad (x \in [0, \pi])$$
  2. **展开为正弦级数（奇延拓）**：
     奇延拓后只计算 $b_n$。
     $$b_n = \frac{2}{\pi} \int_0^{\pi} x \sin nx dx = \frac{2}{\pi} \left( \left[ -\frac{x \cos nx}{n} \right]_0^{\pi} + \int_0^{\pi} \frac{\cos nx}{n} dx \right) = \frac{2}{\pi} \left( -\frac{\pi (-1)^n}{n} \right) = \frac{2 (-1)^{n-1}}{n}$$
     因此正弦级数展开为：
     $$x \sim 2 \sum_{n=1}^{\infty} \frac{(-1)^{n-1}}{n} \sin nx \quad (x \in [0, \pi))$$
     * **避错提示**：正弦展开在 $x = \pi$ 处由于每一项都是 $\sin n\pi = 0$，级数收敛于 0。而原函数值 $f(\pi) = \pi$，故此时在端点处等号不成立。因此必须注明区间为 $[0, \pi)$。

## 四、 PPT例题详解：§3 任意区间与一般周期

### 例 4.1（一般周期分段函数展开）
**【题目】** 设周期为 2 的函数 $f(x)$ 在 $[-1, 1]$ 上的定义为：
$$f(x) = \begin{cases} x & x \in [-1, 0] \\ 1 & x \in (0, 1] \end{cases}$$
求其傅里叶级数，并讨论在各点处的收敛值。
* **考点**：任意周期 $2l = 2 \implies l = 1$ 的傅里叶展开与 Dirichlet 定理。
* **解析**：
  1. **计算系数 $a_0$**（由于 $l=1$）：
     $$a_0 = \frac{1}{1} \int_{-1}^1 f(x) dx = \int_{-1}^0 x dx + \int_0^1 1 dx = \left[ \frac{x^2}{2} \right]_{-1}^0 + 1 = -\frac{1}{2} + 1 = \frac{1}{2}$$
  2. **计算系数 $a_n$**：
     $$a_n = \int_{-1}^1 f(x) \cos n\pi x dx = \int_{-1}^0 x \cos n\pi x dx + \int_0^1 \cos n\pi x dx$$
     由于第二项 $\int_0^1 \cos n\pi x dx = \left[ \frac{\sin n\pi x}{n\pi} \right]_0^1 = 0$：
     $$a_n = \int_{-1}^0 x \cos n\pi x dx = \left[ \frac{x \sin n\pi x}{n\pi} \right]_{-1}^0 - \int_{-1}^0 \frac{\sin n\pi x}{n\pi} dx = 0 + \left[ \frac{\cos n\pi x}{(n\pi)^2} \right]_{-1}^0 = \frac{1 - (-1)^n}{(n\pi)^2}$$
  3. **计算系数 $b_n$**：
     $$b_n = \int_{-1}^1 f(x) \sin n\pi x dx = \int_{-1}^0 x \sin n\pi x dx + \int_0^1 \sin n\pi x dx$$
     * $\int_0^1 \sin n\pi x dx = \left[ -\frac{\cos n\pi x}{n\pi} \right]_0^1 = \frac{1 - (-1)^n}{n\pi}$；
     * $\int_{-1}^0 x \sin n\pi x dx = \left[ -\frac{x \cos n\pi x}{n\pi} \right]_{-1}^0 + \int_{-1}^0 \frac{\cos n\pi x}{n\pi} dx = -\frac{(-1)^n}{n\pi} + 0 = -\frac{(-1)^n}{n\pi}$。
     合并得：
     $$b_n = \frac{1 - 2(-1)^n}{n\pi}$$
  4. **傅里叶级数**：
     $$f(x) \sim \frac{1}{4} + \sum_{n=1}^{\infty} \left[ \frac{1 - (-1)^n}{(n\pi)^2} \cos n\pi x + \frac{1 - 2(-1)^n}{n\pi} \sin n\pi x \right]$$
  5. **收敛性分析**：
     * 当 $x \in (-1, 0) \cup (0, 1)$ 时，函数连续，级数收敛于 $f(x)$；
     * 在跳跃点 $x = 0$ 处，级数收敛于：
       $$\bar{S}(0) = \frac{f(0^+) + f(0^-)}{2} = \frac{1 + 0}{2} = \frac{1}{2}$$
     * 在周期边界点 $x = \pm 1$ 处，级数收敛于：
       $$\bar{S}(\pm 1) = \frac{f(-1^+) + f(1^-)}{2} = \frac{-1 + 1}{2} = 0$$

### 例 5.1（半区间余弦展开的应用）
**【题目】** 将函数 $f(x) = \frac{\pi - x}{2}$ 在 $[0, \pi]$ 上按余弦级数展开。
* **考点**：半区间余弦展开计算。
* **解析**：
  按余弦展开对应进行偶延拓，只需计算 $a_0$ 与 $a_n$。
  1. **计算 $a_0$**：
     $$a_0 = \frac{2}{\pi} \int_0^{\pi} \frac{\pi - x}{2} dx = \frac{1}{\pi} \left[ \pi x - \frac{x^2}{2} \right]_0^{\pi} = \frac{1}{\pi} \left( \pi^2 - \frac{\pi^2}{2} \right) = \frac{\pi}{2}$$
  2. **计算 $a_n$**：
     $$a_n = \frac{2}{\pi} \int_0^{\pi} \frac{\pi - x}{2} \cos nx dx = \frac{1}{\pi} \int_0^{\pi} (\pi - x) \cos nx dx$$
     使用分部积分法：
     $$a_n = \frac{1}{\pi} \left( \left[ \frac{(\pi-x)\sin nx}{n} \right]_0^{\pi} + \int_0^{\pi} \frac{\sin nx}{n} dx \right) = \frac{1}{n\pi} \left[ -\frac{\cos nx}{n} \right]_0^{\pi} = \frac{1 - (-1)^n}{\pi n^2}$$
     * 当 $n$ 为偶数时，$a_n = 0$；
     * 当 $n$ 为奇数时，$a_n = \frac{2}{\pi n^2}$。
  3. **傅里叶余弦级数**：
     由于偶延拓后函数在整个区间上连续，级数在 $[0, \pi]$ 上处处收敛于 $f(x)$：
     $$\frac{\pi - x}{2} = \frac{\pi}{4} + \frac{2}{\pi} \sum_{k=1}^{\infty} \frac{\cos(2k-1)x}{(2k-1)^2} \quad (x \in [0, \pi])$$

### 例 5.2（半区间正弦级数的高阶展开）
**【题目】** 将函数 $f(x) = x(\pi - x)$ 在 $[0, \pi]$ 上按正弦级数展开。
* **考点**：正弦级数展开与分部积分技术。
* **解析**：
  按正弦展开对应奇延拓，只需计算 $b_n$。
  1. **计算 $b_n$**：
     $$b_n = \frac{2}{\pi} \int_0^{\pi} x(\pi - x) \sin nx dx = 2 \int_0^{\pi} x \sin nx dx - \frac{2}{\pi} \int_0^{\pi} x^2 \sin nx dx$$
     我们分别计算两个积分：
     * $I_1 = \int_0^{\pi} x \sin nx dx = \left[ -\frac{x \cos nx}{n} \right]_0^{\pi} + \int_0^{\pi} \frac{\cos nx}{n} dx = -\frac{\pi (-1)^n}{n}$
     * $I_2 = \int_0^{\pi} x^2 \sin nx dx = \left[ -\frac{x^2 \cos nx}{n} \right]_0^{\pi} + \frac{2}{n} \int_0^{\pi} x \cos nx dx$
       而 $\int_0^{\pi} x \cos nx dx = \left[ \frac{x \sin nx}{n} \right]_0^{\pi} - \int_0^{\pi} \frac{\sin nx}{n} dx = \frac{(-1)^n - 1}{n^2}$。
       代入得：
       $$I_2 = -\frac{\pi^2 (-1)^n}{n} + \frac{2((-1)^n - 1)}{n^3}$$
     代回原式合并：
     $$b_n = 2 \left( -\frac{\pi (-1)^n}{n} \right) - \frac{2}{\pi} \left( -\frac{\pi^2 (-1)^n}{n} + \frac{2((-1)^n - 1)}{n^3} \right)$$
     $$b_n = -\frac{2\pi (-1)^n}{n} + \frac{2\pi (-1)^n}{n} - \frac{4((-1)^n - 1)}{\pi n^3} = \frac{4 (1 - (-1)^n)}{\pi n^3}$$
     * 当 $n$ 为偶数时，$b_n = 0$；
     * 当 $n$ 为奇数时，$b_n = \frac{8}{\pi n^3}$。
  2. **傅里叶正弦级数**：
     由于奇延拓后函数在端点处（0 和 $\pi$）的值均为 0，刚好与级数收敛值吻合，因此区间可以取闭区间 $[0, \pi]$：
     $$x(\pi - x) = \frac{8}{\pi} \sum_{k=1}^{\infty} \frac{\sin(2k-1)x}{(2k-1)^3} \quad (x \in [0, \pi])$$

## 五、 备考解题套路与易错防范

### 1. 收敛定理中“等号”的严谨写法
在书写 Fourier 级数时，如果题目没有特殊要求，最好写 $\sim$（代表“对应傅里叶级数”）。如果要写 $=$（等号），必须指明其对应的连续区间，且在间断点及端点处明确指出其收敛值。

### 2. 积分计算的对称性利用
如果区间是对称的 $[-l, l]$：
* 奇函数 $\times$ 奇函数 = 偶函数；
* 偶函数 $\times$ 偶函数 = 偶函数；
* 奇函数 $\times$ 偶函数 = 奇函数。
通过判断被积函数的奇偶性，可以避免极其繁琐的 $a_n$ 或 $b_n$ 积分计算。
