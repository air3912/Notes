# 工科数学分析期末大题突破：第 21 章 曲线积分与曲面积分（PPT例题与详解）

> **期末核心导读**：曲线积分与曲面积分是工科数学分析下册的难点。核心大题考查点包含：**利用球面的对称性求第一型曲线/曲面积分**，**第二型曲线积分的参数化与方向判定（包括功的计算）**，以及**第二型曲面积分的投影计算（方向侧的投影微元符号）**。

## 二、 核心公式与计算步骤（必背）

### 1. 第一型与第二型曲线积分的换算
* **第一型（弧长积分）**：
  $$\int_L f(x,y,z) ds = \int_a^b f(x(t), y(t), z(t)) \sqrt{[x'(t)]^2 + [y'(t)]^2 + [z'(t)]^2} dt$$
* **第二型（坐标积分）**：
  $$\int_L Pdx + Qdy + Rdz = \int_a^b \left[ P(t)x'(t) + Q(t)y'(t) + R(t)z'(t) \right] dt$$

### 2. 第二型曲面积分的投影计算法
对于曲面 $\Sigma: z = z(x,y)$，投影到 $xy$ 平面上的区域为 $D$：
* 若取 **上侧**（法向量 $z$ 分量为正），则面积因子为 $(-z_x, -z_y, 1)$：
  $$\iint_{\Sigma} P dydz + Q dzdx + R dxdy = \iint_D \left[ -P z_x - Q z_y + R \right] dxdy$$
* 若取 **下侧**（法向量 $z$ 分量为负），则取相反号：
  $$\iint_{\Sigma} P dydz + Q dzdx + R dxdy = \iint_D \left[ P z_x + Q z_y - R \right] dxdy$$

## 三、 PPT例题详解：§1 第一型曲线与曲面积分

### 例 1.3（空间曲线积分的对称性秒杀法）
**【题目】** 计算第一型曲线积分 $I = \int_L (x^2+y^2+2z) ds$，其中 $L$ 为球面 $x^2+y^2+z^2 = a^2$ 与平面 $x+y+z = 0$ 的交线。
* **考点**：球面交线积分，对称性化简（期末大题高分捷径）。
* **解析**：
  因为 $L$ 是球面与经过原点平面的交线，所以 $L$ 是一个中心在原点、半径为 $a$ 的大圆。其周长为 $2\pi a$，即 $\int_L ds = 2\pi a$。
  1. **简化被积函数**：
     在 $L$ 上，由于点在球面上，满足 $x^2+y^2+z^2 = a^2 \implies x^2+y^2 = a^2 - z^2$。
     代入被积式：
     $$I = \int_L (a^2 - z^2 + 2z) ds = a^2 \int_L ds - \int_L z^2 ds + 2 \int_L z ds$$
  2. **利用轮换对称性简化积分**：
     * 由于 $L$ 关于原点对称，且平面方程为 $x+y+z = 0 \implies \int_L (x+y+z) ds = 0$。
       由轮换对称性，$\int_L x ds = \int_L y ds = \int_L z ds$，所以：
       $$3 \int_L z ds = 0 \implies \int_L z ds = 0$$
     * 同样，对于二次项，由于球面与平面的对称性，有 $\int_L x^2 ds = \int_L y^2 ds = \int_L z^2 ds$。
       因为在 $L$ 上 $x^2+y^2+z^2 = a^2$，有：
       $$\int_L (x^2+y^2+z^2) ds = \int_L a^2 ds = a^2 (2\pi a) = 2\pi a^3$$
       $$3 \int_L z^2 ds = 2\pi a^3 \implies \int_L z^2 ds = \frac{2}{3} \pi a^3$$
  3. **代回求出结果**：
     $$I = a^2 (2\pi a) - \frac{2}{3} \pi a^3 + 0 = \frac{4}{3} \pi a^3$$

## 四、 PPT例题详解：§2 第二型曲线与曲面积分

### 例 1.3（第一卦限交线积分与轮换对称）
**【题目】** 计算曲线积分 $\int_L (y^2 - z^2) dx + (z^2 - x^2) dy + (x^2 - y^2) dz$，其中 $L$ 为球面 $x^2+y^2+z^2=1$ 在第一卦限部分的边界，从球面外部看为顺时针方向。
* **考点**：分段空间曲线的第二型积分，方向与对称性。
* **解析**：
  边界 $L$ 由位于三个坐标面上的三段四分之一圆弧 $L_1, L_2, L_3$ 组成。
  1. **分析 $L_1$**：在 $xy$ 平面上，$z = 0, dz = 0$。圆弧方程为 $x^2+y^2 = 1$。
     由于整体方向从外面看是顺时针，故 $L_1$ 是从 $(1,0,0)$ 沿着圆弧走到 $(0,1,0)$ 的反向，即从 $(0,1,0)$ 到 $(1,0,0)$。
     积分式为：
     $$I_1 = \int_{L_1} y^2 dx - x^2 dy$$
     参数化：$x = \cos t, y = \sin t$，其中 $t$ 从 $\frac{\pi}{2}$ 到 $0$：
     $$I_1 = \int_{\pi/2}^0 \left[ \sin^2 t (-\sin t) - \cos^2 t (\cos t) \right] dt = \int_0^{\pi/2} (\sin^3 t + \cos^3 t) dt$$
     已知 $\int_0^{\pi/2} \sin^3 t dt = \int_0^{\pi/2} \cos^3 t dt = \frac{2}{3}$，所以：
     $$I_1 = \frac{2}{3} + \frac{2}{3} = \frac{4}{3}$$
  2. **利用循环对称性**：
     由于被积函数和积分区域关于 $x, y, z$ 具有循环轮换对称性，另外两段弧上的积分完全相同：
     $$I_2 = I_3 = \frac{4}{3}$$
  3. **计算总积分**：
     $$I = I_1 + I_2 + I_3 = 3 \times \frac{4}{3} = 4$$

### 例 3.3（投影法求抛物面通量）
**【题目】** 计算 $\iint_{\Sigma} (x^2+z) dydz + \sqrt{z} dxdy$，其中 $\Sigma$ 为抛物面 $z = x^2+y^2$ 在平面 $z=0$ 与 $z=2$ 之间的部分，方向取下侧。
* **考点**：第二型曲面积分投影公式，奇偶对称性。
* **解析**：
  1. **使用投影公式转化**：
     已知曲面方程为 $z = x^2+y^2$，其在 $xy$ 平面上的投影区域 $D$ 为圆盘 $x^2+y^2 \le 2$。
     曲面取 **下侧**，所以对应的面积因子为：
     $$\vec{n} dS = (z_x, z_y, -1) dxdy = (2x, 2y, -1) dxdy$$
     被积项为 $P = x^2+z = 2x^2+y^2$，$Q = 0$，$R = \sqrt{z} = \sqrt{x^2+y^2}$。
     代入投影公式：
     $$I = \iint_D \left[ (2x^2+y^2)(2x) + 0 \cdot (2y) - \sqrt{x^2+y^2} \right] dxdy$$
     $$I = \iint_D \left[ 4x^3 + 2xy^2 - \sqrt{x^2+y^2} \right] dxdy$$
  2. **利用对称性消去奇函数项**：
     因为投影区域 $D: x^2+y^2 \le 2$ 是关于 $y$ 轴对称的，而被积函数中的 $4x^3$ 和 $2xy^2$ 关于 $x$ 是奇函数，所以在对称域上的积分为 0：
     $$\iint_D (4x^3 + 2xy^2) dxdy = 0$$
  3. **利用极坐标计算余下部分**：
     $$I = -\iint_D \sqrt{x^2+y^2} dxdy$$
     引入极坐标 $x = r\cos\theta, y = r\sin\theta$，其中 $0 \le r \le \sqrt{2}, 0 \le \theta \le 2\pi$：
     $$I = -\int_0^{2\pi} d\theta \int_0^{\sqrt{2}} r \cdot r dr = -2\pi \left[ \frac{r^3}{3} \right]_0^{\sqrt{2}} = -\frac{4\sqrt{2}}{3} \pi$$

## 五、 备考解题套路与易错防范

### 1. 牢记第二型曲面积分投影时的“负号”
很多同学会直接把 $dxdy$ 替换为投影域的积分，而忽视了曲面法向量的方向。
* **上侧**：$R$ 项取 **正** ($+R dxdy$)；
* **下侧**：$R$ 项取 **负** ($-R dxdy$)。
对于 $dydz$ 和 $dzdx$ 的投影也同样存在与法向量指向（前/后、左/右）对应的符号变化。

### 2. 巧用对称性秒杀第一型积分
第一型积分（如 $\int_L z ds$）没有方向。如果积分域关于某个坐标轴或平面对称，且被积函数关于该变量是奇函数，那么这一项的积分值直接就是 0，省去了所有积分代换的复杂步骤。
