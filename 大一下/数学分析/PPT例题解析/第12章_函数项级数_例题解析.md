# 工科数学分析期末大题突破：第 12 章 函数项级数（PPT例题与详解）

> **期末核心导读**：函数项级数的一致收敛性是工科数学分析中的必考理论大题。主要的考核形式包括：**利用 $M$-判别法或 Dirichlet/Abel 判别法判定一致收敛性**、**通过构造特殊序列点证明非一致收敛性**，以及**利用逐项求导、逐项积分性质求解级数的和函数或证明分析性质**。

---

## 一、 考情分析与考查频率

| 大题考点 | 常见考查形式 | 考查频率 |
| :--- | :--- | :---: |
| **一致收敛判定（Weierstrass $M$ 判别）** | 用 $M$-判别法证明级数一致收敛，常配合幂级数或指数衰减项 | ★★★★★ |
| **非一致收敛判定（反例/点序列测试）** | 讨论 $S_n(x)$ 在特定区间上不一致收敛，通过取 $x_n$ 使得 $|S_n(x_n) - S(x_n)| \ge c > 0$ | ★★★★★ |

| **Dini 定理与性质保留反例** | 考查点态极限不保留连续性、可导性、可积性的反例，或使用 Dini 定理判定 | ★★☆☆☆ |

## 二、 核心公式与判定定理（必背）

### 1. 一致收敛的定义与等价判定
* **定义**：设 $\{S_n(x)\}$ 在区间 $D$ 上收敛于极限函数 $S(x)$。若对于任意给定的 $\epsilon > 0$，存在与 $x$ 无关的 $N$，使得当 $n > N$ 时，对一切 $x \in D$ 均有 $|S_n(x) - S(x)| < \epsilon$，则称 $\{S_n(x)\}$ 在 $D$ 上一致收敛于 $S(x)$。
* **Supremum 测试法**：
  $$S_n(x) \to S(x) \text{ 一致收敛于 } D \quad \Longleftrightarrow \quad \lim_{n \to \infty} \sup_{x \in D} |S_n(x) - S(x)| = 0$$
* **必要条件**：若级数 $\sum u_n(x)$ 一致收敛，则通项 $u_n(x)$ 必在 $D$ 上一致趋于 0。

### 2. 核心判定定理
* **Weierstrass $M$-判别法**（最常用）：
  若在 $D$ 上对所有 $n$ 有 $|u_n(x)| \le M_n$，且常数项级数 $\sum M_n$ 收敛，则 $\sum u_n(x)$ 在 $D$ 上一致收敛（且绝对收敛）。
* **Dirichlet 一致收敛判别法**：
  若级数 $\sum a_n(x)$ 的部分和函数序列在 $D$ 上一致有界，而 $b_n(x)$ 对于每个 $x \in D$ 关于 $n$ 单调，且在 $D$ 上 $b_n(x) \to 0$ 一致收敛，则 $\sum a_n(x)b_n(x)$ 在 $D$ 上一致收敛。
* **Abel 一致收敛判别法**：
  若级数 $\sum a_n(x)$ 在 $D$ 上一致收敛，而 $b_n(x)$ 对于每个 $x \in D$ 关于 $n$ 单调，且在 $D$ 上一致有界，则 $\sum a_n(x)b_n(x)$ 在 $D$ 上一致收敛。

### 3. 一致收敛的分析性质
* **连续性**：若 $u_n(x)$ 连续且级数一致收敛，则和函数 $S(x)$ 连续。
* **逐项积分**：若级数在 $[a,b]$ 上一致收敛，可将积分与求和号交换：
  $$\int_a^b \sum u_n(x) dx = \sum \int_a^b u_n(x) dx$$
* **逐项求导**：若 $u_n(x)$ 可导，$\sum u_n(x_0)$ 收敛，且**导数级数 $\sum u_n'(x)$ 一致收敛**，则和函数可导且可逐项求导：
  $$\left( \sum u_n(x) \right)' = \sum u_n'(x)$$

## 三、 PPT例题详解：§1 函数序列一致收敛概念

### 例 2.1 & 2.2（可导性不保留的反例）
**【题目】** 设 $S_n(x) = \sqrt{x^2 + \frac{1}{n^2}}$，讨论 $\{S_n(x)\}$ 在 $\mathbb{R}$ 上的敛散性与可导性。
* **考点**：一致收敛但极限函数不可导的经典反例。
* **解析**：
  1. **求点态极限**：
     $$S(x) = \lim_{n \to \infty} \sqrt{x^2 + \frac{1}{n^2}} = \sqrt{x^2} = |x|$$
  2. **证明一致收敛**：
     $$|S_n(x) - S(x)| = \sqrt{x^2 + \frac{1}{n^2}} - |x| = \frac{\frac{1}{n^2}}{\sqrt{x^2 + \frac{1}{n^2}} + |x|} \le \frac{\frac{1}{n^2}}{\frac{1}{n}} = \frac{1}{n}$$
     对于任意 $x \in \mathbb{R}$，上式均成立。
     由于 $\lim_{n \to \infty} \sup_{x \in \mathbb{R}} |S_n(x) - S(x)| \le \lim_{n \to \infty} \frac{1}{n} = 0$，
     故 $\{S_n(x)\}$ 在整个实数集 $\mathbb{R}$ 上一致收敛于 $S(x) = |x|$。
  3. **可导性分析**：
     每个 $S_n(x)$ 在 $\mathbb{R}$ 上显然是处处可导的。但是其一致收敛的极限函数 $S(x) = |x|$ 在 $x = 0$ 处却不可导。
     * **物理意义**：即使一致收敛，也不能保证极限函数保留可导性，除非导数级数也满足一致收敛。

### 例 2.3 & 2.4（可积性与极限交换不成立的反例）
**【题目】** 设 $S_n(x) = n x (1-x^2)^n$，讨论其在区间 $[0, 1]$ 上的极限，并比较 $\lim_{n\to\infty} \int_0^1 S_n(x) dx$ 与 $\int_0^1 \lim_{n\to\infty} S_n(x) dx$ 是否相等。
* **考点**：积分与极限运算交换顺序失效的反例。
* **解析**：
  1. **求点态极限**：
     * 当 $x = 0$ 时，$S_n(0) = 0 \implies \lim_{n\to\infty} S_n(0) = 0$。
     * 当 $x \in (0, 1]$ 时，由于 $0 \le 1-x^2 < 1$，由极限阶的性质（指数衰减快于多项式增长），有 $\lim_{n\to\infty} n (1-x^2)^n = 0$，故 $\lim_{n\to\infty} S_n(x) = 0$。
     * 因此，极限函数 $S(x) \equiv 0$。从而：
       $$\int_0^1 \lim_{n\to\infty} S_n(x) dx = \int_0^1 0 \, dx = 0$$
  2. **求积分的极限**：
     $$\int_0^1 S_n(x) dx = \int_0^1 n x (1-x^2)^n dx = \left[ -\frac{n}{2(n+1)} (1-x^2)^{n+1} \right]_0^1 = \frac{n}{2(n+1)}$$
     求其极限：
     $$\lim_{n\to\infty} \int_0^1 S_n(x) dx = \lim_{n\to\infty} \frac{n}{2(n+1)} = \frac{1}{2}$$
  3. **结论**：
     $$\lim_{n\to\infty} \int_0^1 S_n(x) dx = \frac{1}{2} \ne 0 = \int_0^1 \lim_{n\to\infty} S_n(x) dx$$
     因为极限运算次序不可交换，所以 $\{S_n(x)\}$ 在 $[0, 1]$ 上必定**非一致收敛**。

### 例 3.1（利用求导求 Sup 证一致收敛）
**【题目】** 考察 $S_n(x) = \frac{x}{1+n^2 x^2}$ 在 $[0, 1]$ 上的收敛性。
* **考点**：导数法求最大偏差（Supremum 测试法）。
* **解析**：
  1. **求点态极限**：对任意 $x \in [0, 1]$，显然有 $\lim_{n\to\infty} S_n(x) = 0$，即极限函数 $S(x) \equiv 0$。
  2. **计算最大值**：设误差函数 $d_n(x) = |S_n(x) - S(x)| = \frac{x}{1+n^2 x^2}$ (因为 $x \ge 0$)。
     求导数：
     $$d_n'(x) = \frac{1 \cdot (1+n^2 x^2) - x \cdot 2n^2 x}{(1+n^2 x^2)^2} = \frac{1 - n^2 x^2}{(1+n^2 x^2)^2}$$
     令 $d_n'(x) = 0 \implies x_n = \frac{1}{n} \in [0, 1]$（对于 $n \ge 1$）。
     因为当 $x < \frac{1}{n}$ 时导数大于 0，当 $x > \frac{1}{n}$ 时导数小于 0，所以 $x_n = \frac{1}{n}$ 为极大值点，也是最大值点。
     最大偏差为：
     $$\sup_{x \in [0, 1]} |S_n(x) - S(x)| = d_n\left(\frac{1}{n}\right) = \frac{1/n}{1 + n^2 (1/n^2)} = \frac{1}{2n}$$
  3. **判定**：
     $$\lim_{n\to\infty} \sup_{x \in [0, 1]} |S_n(x) - S(x)| = \lim_{n\to\infty} \frac{1}{2n} = 0$$
     所以 $\{S_n(x)\}$ 在 $[0, 1]$ 上一致收敛于 0。

---

### 例 3.2（边界点的非一致收敛分析）
**【题目】** 讨论 $S_n(x) = x^n$ 在区间 $[0, 1)$ 上的一致收敛性。
* **考点**：开区间端点附近引起的不一致收敛。
* **解析**：
  1. **点态极限**：对任意 $x \in [0, 1)$，由于 $|x| < 1$，故 $S(x) = \lim_{n\to\infty} x^n = 0$。
  2. **最大偏差**：
     $$\sup_{x \in [0, 1)} |S_n(x) - S(x)| = \sup_{x \in [0, 1)} x^n = 1$$
     由于 $\lim_{n\to\infty} 1 = 1 \ne 0$，根据 Supremum 测试法，在 $[0, 1)$ 上该序列非一致收敛。
  * **避错提示**：许多学生误认为只要区间不包含端点 1 就能一致收敛。实际上，由于上限可以无限逼近 1，最大偏差始终能达到 1。

### 例 3.3（非一致收敛判定：无界区间）
**【题目】** 设 $S_n(x) = \frac{nx}{1+n^2 x^2}$，考察其在 $(0, +\infty)$ 上的收敛性。
* **考点**：Supremum 测试在无界区间中的应用。
* **解析**：
  1. **求点态极限**：
     * 若 $x > 0$，$\lim_{n\to\infty} S_n(x) = \lim_{n\to\infty} \frac{nx}{n^2 x^2} = 0$。故极限函数 $S(x) \equiv 0$。
  2. **求最大偏差**：
     误差函数为 $d_n(x) = \frac{nx}{1+n^2 x^2}$。其最大值点仍在 $x_n = \frac{1}{n} \in (0, +\infty)$。
     最大值为：
     $$\sup_{x \in (0, +\infty)} |S_n(x) - S(x)| = d_n\left(\frac{1}{n}\right) = \frac{1}{2}$$
  3. **判定**：
     $$\lim_{n\to\infty} \sup_{x \in (0, +\infty)} |S_n(x) - S(x)| = \frac{1}{2} \ne 0$$
     所以该序列在 $(0, +\infty)$ 上非一致收敛。

### 例 3.4 & 3.5（闭区间上的一致收敛区间收缩）
**【题目】** 设 $S_n(x) = (1-x)x^n$：
1. 讨论其在 $[0, 1]$ 上的收敛性；
2. 讨论其在任意 $[0, a]$ ($0 < a < 1$) 上的收敛性。
* **考点**：区间收缩改善收敛性质的典型例题。
* **解析**：
  1. **在 $[0, 1]$ 上的讨论**：
     点态极限显然是 $S(x) = 0$（因为当 $x=1$ 时 $S_n(1)=0$，当 $x<1$ 时 $x^n \to 0$）。
     求最大偏差：误差函数为 $d_n(x) = (1-x)x^n$。
     $$d_n'(x) = n x^{n-1} - (n+1) x^n = x^{n-1} [n - (n+1)x] = 0 \implies x_n = \frac{n}{n+1} \in [0, 1]$$
     最大偏差为：
     $$\sup_{x \in [0, 1]} |S_n(x) - S(x)| = \left(1 - \frac{n}{n+1}\right)\left(\frac{n}{n+1}\right)^n = \frac{1}{n+1}\left(1 - \frac{1}{n+1}\right)^n$$
     求极限：
     $$\lim_{n\to\infty} \frac{1}{n+1}\left(1 - \frac{1}{n+1}\right)^n = 0 \cdot e^{-1} = 0$$
     故在 $[0, 1]$ 上一致收敛。
  2. **在 $[0, a]$ 上的讨论（$0 < a < 1$）**：
     由于 $0 < a < 1$，当 $n$ 充分大时，极大值点 $x_n = \frac{n}{n+1} > a$。
     因此在区间 $[0, a]$ 上，函数 $d_n(x) = (1-x)x^n$ 是单调递增的。
     所以其最大偏差在右端点 $x=a$ 处取得：
     $$\sup_{x \in [0, a]} |S_n(x) - S(x)| = (1-a)a^n$$
     由于 $0 < a < 1$，$\lim_{n\to\infty} (1-a)a^n = 0$。
     故在 $[0, a]$ 上亦一致收敛。

### 例 3.6（非一致收敛：极值点爆炸）
**【题目】** 给定序列 $S_n(x) = n^2 x (1-x^2)^n$，在区间 $D=[0, 1]$ 上判断一致收敛性。
* **考点**：极值点数值爆炸引发的非一致收敛。
* **解析**：
  点态极限为 $S(x) \equiv 0$。
  求极大值点：设 $d_n(x) = n^2 x (1-x^2)^n$。
  $$d_n'(x) = n^2 (1-x^2)^{n-1} [1 - (2n+1)x^2] = 0 \implies x_n = \frac{1}{\sqrt{2n+1}}$$
  在此点处的误差值为：
  $$d_n(x_n) = n^2 \frac{1}{\sqrt{2n+1}} \left( 1 - \frac{1}{2n+1} \right)^n = \frac{n^2}{\sqrt{2n+1}} \left( \frac{2n}{2n+1} \right)^n \sim \frac{n^2}{\sqrt{2n}} e^{-1/2} \to +\infty$$
  由于最大值趋于无穷大，显然非一致收敛。

### 例 3.7 & 3.8（Euler 数列的一致收敛范围）
**【题目】** 设 $S_n(x) = \left(1 + \frac{x}{n}\right)^n$，证明 $\{S_n(x)\}$ 在 $[0, +\infty)$ 上非一致收敛，但在任意 $[0, a]$（$a>0$）上一致收敛。
* **考点**：指数函数的逼近性质。
* **解析**：
  已知点态极限为 $S(x) = e^x$。
  1. **在 $[0, +\infty)$ 上非一致收敛**：
     构造特殊序列点：令 $x_n = n$。
     $$|S_n(x_n) - S(x_n)| = |e^n - (1 + 1)^n| = e^n - 2^n = (e-2)^n \to +\infty \ne 0$$
     所以极限不为 0，在 $[0, +\infty)$ 上非一致收敛。
  2. **在 $[0, a]$ 上一致收敛**：
     利用泰勒展开的误差估计，当 $x \in [0, a]$ 时，可以证明：
     $$0 \le e^x - \left(1 + \frac{x}{n}\right)^n \le \frac{x^2 e^x}{2n} \le \frac{a^2 e^a}{2n}$$
     由于 $\lim_{n\to\infty} \frac{a^2 e^a}{2n} = 0$，故在该有界区间上一致收敛。

## 四、 PPT例题详解：§2 一致收敛判别法

### 例 2.2（Weierstrass $M$ 判别法应用）
**【题目】** 证明函数项级数 $\sum_{n=1}^{\infty} x^a e^{-nx}$ ($a > 1$) 在 $[0, +\infty)$ 上一致收敛。
* **考点**：$M$-判别法，求函数最大值。
* **解析**：
  设通项为 $u_n(x) = x^a e^{-nx}$。由于 $x \ge 0$，有 $u_n(x) \ge 0$。
  为了找到控制级数的常数项级数，我们对通项求极大值：
  $$u_n'(x) = a x^{a-1} e^{-nx} - n x^a e^{-nx} = x^{a-1} e^{-nx} (a - nx) = 0 \implies x_n = \frac{a}{n}$$
  当 $x < \frac{a}{n}$ 时导数大于 0，当 $x > \frac{a}{n}$ 时导数小于 0，因此最大值为：
  $$M_n = u_n\left(\frac{a}{n}\right) = \left(\frac{a}{n}\right)^a e^{-a} = \frac{a^a e^{-a}}{n^a}$$
  这样，在 $[0, +\infty)$ 上对所有 $n$ 均有：
  $$|u_n(x)| \le M_n$$
  考察常数项级数 $\sum_{n=1}^{\infty} M_n = a^a e^{-a} \sum_{n=1}^{\infty} \frac{1}{n^a}$。
  因为已知 $a > 1$，所以常数项基准 $p$-级数 $\sum \frac{1}{n^a}$ 收敛。
  根据 **Weierstrass $M$-判别法**，原级数在 $[0, +\infty)$ 上一致收敛。

### 例 2.3（用连续性反证非一致收敛）
**【题目】** 讨论 $\sum_{n=1}^{\infty} x^a e^{-nx}$ ($0 < a \le 1$) 在 $[0, +\infty)$ 上的收敛性与一致收敛性。
* **考点**：和函数的连续性判定一致收敛性（经典理论考题）。
* **解析**：
  1. **收敛性**：当 $x=0$ 时，各项为 0，级数显然收敛；当 $x > 0$ 时，通项以几何级数的速度衰减，必收敛。
  2. **求和函数**：对 $x > 0$，级数是首项为 $x^a e^{-x}$，公比为 $e^{-x}$ 的几何级数：
     $$S(x) = \sum_{n=1}^{\infty} x^a (e^{-x})^n = x^a \frac{e^{-x}}{1 - e^{-x}} = \frac{x^a}{e^x - 1}$$
     所以和函数为：
     $$S(x) = \begin{cases} \frac{x^a}{e^x - 1} & x > 0 \\ 0 & x = 0 \end{cases}$$
  3. **考查和函数的连续性**：
     我们探究 $S(x)$ 在 $x = 0$ 处的右极限：
     $$\lim_{x\to 0^+} S(x) = \lim_{x\to 0^+} \frac{x^a}{e^x - 1} = \lim_{x\to 0^+} \frac{x^a}{x} = \lim_{x\to 0^+} x^{a-1}$$
     * 若 $0 < a < 1$，则 $a-1 < 0$，故 $\lim_{x\to 0^+} S(x) = +\infty \ne S(0)$。
     * 若 $a = 1$，则 $\lim_{x\to 0^+} S(x) = 1 \ne S(0)$。
     这说明在 $0 < a \le 1$ 时，和函数 $S(x)$ 在 $x = 0$ 处是**不连续的**。
     由于级数的每一项 $u_n(x) = x^a e^{-nx}$ 在 $[0, +\infty)$ 上都是连续函数，如果该级数一致收敛，其和函数必连续。
     然而和函数在此处不连续，产生矛盾。
     **结论**：原级数在 $[0, +\infty)$ 上非一致收敛。

### 例 3.1（Abel 一致收敛判别法应用）
**【题目】** 设常数项级数 $\sum_{n=1}^{\infty} a_n$ 收敛，证明级数 $\sum_{n=1}^{\infty} a_n x^n$ 在 $[0, 1]$ 上一致收敛。
* **考点**：Abel 一致收敛判别法（阿贝尔第二定理的证明）。
* **解析**：
  我们将通项分解为 $u_n(x) = a_n \cdot x^n$。
  1. 级数 $\sum_{n=1}^{\infty} a_n$ 作为函数项级数（与 $x$ 无关），显然在 $[0, 1]$ 上一致收敛；
  2. 函数序列 $v_n(x) = x^n$ 对每个固定的 $x \in [0, 1]$ 关于 $n$ 都是单调递减的（因为 $x^{n+1} \le x^n$）；
  3. $v_n(x) = x^n$ 在区间 $[0, 1]$ 上一致有界，即对所有 $n$ 和所有 $x \in [0, 1]$，均有 $|x^n| \le 1$。
  根据 **Abel 一致收敛判别法**，级数 $\sum_{n=1}^{\infty} a_n x^n$ 在 $[0, 1]$ 上一致收敛。

## 五、 PPT例题详解：§3 和函数的分析性质

### 例 2.1 & 2.2（逐项积分的应用）
**【题目】** 证明对一切 $x \in (-1, 1)$，成立 $\sum_{n=1}^{\infty} \frac{x^n}{n} = -\ln(1-x)$。
* **考点**：逐项积分定理。
* **解析**：
  我们已知几何级数公式：
  $$\frac{1}{1-t} = \sum_{n=0}^{\infty} t^n \quad (|t| < 1)$$
  对任意 $x \in (-1, 1)$，取闭子区间 $[-r, r]$ 满足 $|x| \le r < 1$。
  因为级数 $\sum_{n=0}^{\infty} t^n$ 在 $[-r, r]$ 上一致收敛（通过 $M$-判别法，有 $|t^n| \le r^n$ 且 $\sum r^n$ 收敛）。
  根据逐项积分定理，可以在区间 $[0, x]$ 上进行积分与求和号的交换：
  $$\int_0^x \frac{dt}{1-t} = \int_0^x \left( \sum_{n=0}^{\infty} t^n \right) dt = \sum_{n=0}^{\infty} \int_0^x t^n dt$$
  左侧积分得：
  $$\int_0^x \frac{dt}{1-t} = \left[ -\ln(1-t) \right]_0^x = -\ln(1-x)$$
  右侧逐项积分为：
  $$\sum_{n=0}^{\infty} \frac{x^{n+1}}{n+1} = \sum_{n=1}^{\infty} \frac{x^n}{n}$$
  因此结论成立：对于所有 $x \in (-1, 1)$，$\sum_{n=1}^{\infty} \frac{x^n}{n} = -\ln(1-x)$。

### 例 3.1（逐项求导的应用）
**【题目】** 证明对一切 $x \in (-1, 1)$，成立 $\sum_{n=1}^{\infty} n x^n = \frac{x}{(1-x)^2}$。
* **考点**：逐项求导定理。
* **解析**：
  从几何级数公式出发：
  $$\frac{1}{1-x} = \sum_{n=0}^{\infty} x^n \quad (|x| < 1)$$
  为了对级数进行逐项求导，必须验证导数级数的一致收敛性。
  对任意 $x \in (-1, 1)$，取闭子区间 $[-r, r]$（$0 < r < 1$）。
  求导后项为 $n x^{n-1}$。在 $[-r, r]$ 上，有：
  $$|n x^{n-1}| \le n r^{n-1}$$
  利用比值法可知，常数项级数 $\sum_{n=1}^{\infty} n r^{n-1}$ 是收敛的（比值为 $r < 1$）。
  根据 Weierstrass $M$-判别法，导数级数 $\sum_{n=1}^{\infty} n x^{n-1}$ 在 $[-r, r]$ 上一致收敛。
  因此，根据逐项求导定理：
  $$\left( \frac{1}{1-x} \right)' = \left( \sum_{n=0}^{\infty} x^n \right)' = \sum_{n=1}^{\infty} n x^{n-1}$$
  计算左侧导数得：
  $$\frac{1}{(1-x)^2} = \sum_{n=1}^{\infty} n x^{n-1}$$
  两边同乘以 $x$ 即得：
  $$\frac{x}{(1-x)^2} = \sum_{n=1}^{\infty} n x^n$$
  该式在整个区间 $(-1, 1)$ 内均成立。

## 六、 备考解题套路与易错防范

### 1. 证明“一致收敛”的三步法
1. **求出点态极限 $S(x)$**：通常直接让 $n \to \infty$。
2. **写出绝对误差 $|S_n(x) - S(x)|$**（对于级数则是余项 $|R_n(x)|$）。
3. **求误差的最大值（或上确界）**：
   * 在区间内求导，寻找极值点 $x_n$。
   * 计算极值点处的偏差 $d_n(x_n)$。
   * 若 $\lim_{n\to\infty} d_n(x_n) = 0$，则一致收敛；若不趋于 0，则非一致收敛。

### 2. 证明“非一致收敛”的常用抓手
最有力的大题技巧是**构造特殊点序列 $\{x_n\}$**：
寻找一个包含参数 $n$ 的自变量序列 $x_n$（通常是函数的极大值点，如 $x_n = \frac{1}{n}$ 或 $x_n = 1 - \frac{1}{n}$），使得：
$$|S_n(x_n) - S(x_n)| \ge c > 0 \quad (\text{对常数 } c \text{ 成立})$$
即可直接判定非一致收敛。
