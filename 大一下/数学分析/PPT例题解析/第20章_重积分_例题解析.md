# 工科数学分析期末大题突破：第 20 章 重积分（PPT例题与详解）

> **期末核心导读**：重积分是期末考试分值最高的大题考点之一。主要的命题题型包含：**直角坐标下二重积分的换序计算（原顺序无法积出）**，**极坐标及广义极坐标（椭圆变换）下二重积分的计算**，**三重积分在柱坐标与球坐标下的体积/质量/重心计算**，以及**利用第一型曲面积分公式计算曲面面积**。

---

## 一、 考情分析与考查频率

| 大题考点 | 常见考查形式 | 考查频率 |
| :--- | :--- | :---: |
| **二重积分交换积分次序** | 针对被积函数含 $\sin(x^2)$ 或 $e^{x^2}$ 等项，必须通过画图改变积分次序才能求出结果 | ★★★★★ |
| **柱/球坐标系三重积分** | 计算由旋转面（如锥面与球面、抛物面与平面）围成几何体的体积或重心坐标 | ★★★★★ |
| **极坐标与广义极坐标变换** | 计算圆盘、环形域或椭圆域上的积分，利用 $x=ar\cos\theta, y=br\sin\theta$ 简化计算 | ★★★★☆ |
| **曲面面积计算** | 计算显式曲面 $z=f(x,y)$ 或球面在锥体内部部分的面积，使用面积因子 $dS$ | ★★★★☆ |

## 二、 核心公式与坐标代换（必背）

### 1. 二重积分极坐标代换
令 $x = r \cos\theta, y = r \sin\theta$，则面积微元为：
$$dxdy = r dr d\theta$$

### 2. 广义极坐标（椭圆代换）
对于椭圆域 $\frac{x^2}{a^2} + \frac{y^2}{b^2} \le 1$，令 $x = a r \cos\theta, y = b r \sin\theta$，则：
$$dxdy = ab r dr d\theta$$

### 3. 三重积分柱坐标与球坐标代换
* **柱坐标**：$x = r\cos\theta, y = r\sin\theta, z = z$，体积微元为：
  $$dV = r dr d\theta dz$$
* **球坐标**：$x = \rho \sin\varphi \cos\theta, y = \rho \sin\varphi \sin\theta, z = \rho \cos\varphi$，体积微元为：
  $$dV = \rho^2 \sin\varphi d\rho d\varphi d\theta$$

### 4. 曲面面积计算公式
对于曲面 $\Sigma: z = f(x,y)$ 在 $xy$ 平面的投影区域为 $D$，其面积为：
$$S = \iint_D \sqrt{1 + z_x^2 + z_y^2} dxdy$$

## 三、 PPT例题详解：§2 重积分化累次积分

### 例 2.2（改变积分次序化难为易）
**【题目】** 计算二重积分 $I = \iint_D \sin(x^2) dxdy$，其中 $D$ 是由 $y=0$, $x=\sqrt{\frac{\pi}{2}}$, $y=x$ 所围成的闭区域。
* **考点**：交换积分次序以消去无法积出的项（期末经典必考）。
* **解析**：
  1. **画出区域并确定原积分限**：
     区域 $D$ 是以 $(0,0)$、$\left(\sqrt{\frac{\pi}{2}}, 0\right)$ 和 $\left(\sqrt{\frac{\pi}{2}}, \sqrt{\frac{\pi}{2}}\right)$ 为顶点的直角三角形。
     原累次积分如果是先对 $x$ 积分：$\int \sin(x^2) dx$ 是非初等函数（Fresnel 积分），无法积出。
     所以必须写为先对 $y$ 积分的累次积分：
     $$I = \int_0^{\sqrt{\pi/2}} dx \int_0^x \sin(x^2) dy$$
  2. **逐步计算**：
     先计算内层关于 $y$ 的积分（将 $x$ 视为常数）：
     $$\int_0^x \sin(x^2) dy = \left[ y \sin(x^2) \right]_0^x = x \sin(x^2)$$
     再计算外层关于 $x$ 的积分：
     $$I = \int_0^{\sqrt{\pi/2}} x \sin(x^2) dx$$
     令 $u = x^2 \implies du = 2x dx$：
     $$I = \frac{1}{2} \int_0^{\pi/2} \sin u du = \frac{1}{2} \left[ -\cos u \right]_0^{\pi/2} = \frac{1}{2} (0 - (-1)) = \frac{1}{2}$$

## 四、 PPT例题详解：§3 重积分变量代换

### 例 2.3（极坐标求解径向积分）
**【题目】** 计算 $\iint_D \sin\left(\pi \sqrt{x^2+y^2}\right) dxdy$，其中 $D = \{(x,y) \mid x^2+y^2 \le 1\}$。
* **考点**：极坐标在圆形区域上的应用。
* **解析**：
  利用极坐标代换 $x = r\cos\theta, y = r\sin\theta$，积分限变为 $0 \le r \le 1, 0 \le \theta \le 2\pi$：
  $$I = \int_0^{2\pi} d\theta \int_0^1 \sin(\pi r) \cdot r dr = 2\pi \int_0^1 r \sin(\pi r) dr$$
  对内层积分应用分部积分法：
  $$\int_0^1 r \sin(\pi r) dr = \left[ -\frac{r \cos(\pi r)}{\pi} \right]_0^1 - \int_0^1 \left( -\frac{\cos(\pi r)}{\pi} \right) dr$$
  $$\int_0^1 r \sin(\pi r) dr = \left( \frac{1}{\pi} - 0 \right) + \left[ \frac{\sin(\pi r)}{\pi^2} \right]_0^1 = \frac{1}{pi}$$
  所以原动积分值为：
  $$I = 2\pi \cdot \frac{1}{\pi} = 2$$

### 例 2.5（广义极坐标求解双纽线面积）
**【题目】** 求曲线 $\left( \frac{x^2}{a^2} + \frac{y^2}{b^2} \right)^2 = \frac{xy}{c^2}$ ($a, b, c > 0$) 所围图形的面积。
* **考点**：广义极坐标代换（椭圆代换）的运用。
* **解析**：
  由于曲线关于原点对称，且由于要求 $xy \ge 0$，该图形仅分布在第一和第三象限。我们只算第一象限部分的面积，最后乘以 2。
  引入广义极坐标（椭圆代换）：
  $$x = a r \cos\theta, \quad y = b r \sin\theta$$
  其 Jacobian 值为 $J = ab r$。
  1. **将曲线方程转化为极坐标形式**：
     左边 $\left( \frac{x^2}{a^2} + \frac{y^2}{b^2} \right)^2 = (r^2)^2 = r^4$。
     右边 $\frac{xy}{c^2} = \frac{ab r^2 \sin\theta \cos\theta}{c^2} = \frac{ab r^2 \sin 2\theta}{2c^2}$。
     消去 $r^2$ 得曲线方程为：
     $$r^2 = \frac{ab}{2c^2} \sin 2\theta$$
     在第一象限，$\theta \in \left[0, \frac{\pi}{2}\right]$，此时 $\sin 2\theta \ge 0$，围成一个叶瓣。
  2. **计算面积**：
     $$A = 2 \iint_{D_1} dxdy = 2 \int_0^{\pi/2} d\theta \int_0^{\sqrt{\frac{ab}{2c^2} \sin 2\theta}} ab r dr$$
     $$A = 2ab \int_0^{\pi/2} \left[ \frac{r^2}{2} \right]_0^{\sqrt{\frac{ab}{2c^2} \sin 2\theta}} d\theta = ab \int_0^{\pi/2} \frac{ab}{2c^2} \sin 2\theta d\theta$$
     $$A = \frac{a^2 b^2}{2c^2} \left[ -\frac{\cos 2\theta}{2} \right]_0^{\pi/2} = \frac{a^2 b^2}{2c^2} \left( \frac{1}{2} - \left(-\frac{1}{2}\right) \right) = \frac{a^2 b^2}{2c^2}$$

## 五、 PPT例题详解：§4 曲面面积

### 例 1.2（球面在锥面内部的面积）
**【题目】** 设一曲面 $\Sigma$ 为球面 $x^2+y^2+z^2 = 2Rz$ 被包含在锥面 $z^2 = 3(x^2+y^2)$ 内的部分，求该曲面的面积。
* **考点**：图形曲面面积公式，球面投影计算。
* **解析**：
  1. **确定两曲面交线与投影域**：
     将锥面代入球面方程：
     $$(x^2+y^2) + z^2 = 2Rz \implies \frac{z^2}{3} + z^2 = 2Rz \implies \frac{4}{3} z^2 = 2Rz$$
     因为部分在锥面内且位于上方，消去 $z \ne 0$ 得交线高度为 $z = \frac{3}{2} R$。
     此时交线投影到 $xy$ 平面上的圆方程为：
     $$x^2+y^2 = \frac{z^2}{3} = \frac{9R^2/4}{3} = \frac{3}{4} R^2$$
     即投影区域 $D$ 为以原点为圆心、半径为 $r_0 = \frac{\sqrt{3}}{2} R$ 的圆盘。
  2. **写出球面面积微元**：
     将球面写为隐式方程 $F(x,y,z) = x^2+y^2+z^2-2Rz = 0$。其法向量（梯度）为：
     $$\nabla F = (2x, 2y, 2z-2R)$$
     曲面在 $xy$ 平面上的投影面积因子为：
     $$dS = \frac{|\nabla F|}{|F_z|} dxdy = \frac{2\sqrt{x^2+y^2+(z-R)^2}}{2|z-R|} dxdy = \frac{R}{|z-R|} dxdy$$
     由于球面方程可变形为 $x^2+y^2 + (z-R)^2 = R^2 \implies |z-R| = \sqrt{R^2 - (x^2+y^2)}$。
     所以：
     $$dS = \frac{R}{\sqrt{R^2 - (x^2+y^2)}} dxdy$$
  3. **积分求解面积**：
     $$S = \iint_D \frac{R}{\sqrt{R^2 - (x^2+y^2)}} dxdy$$
     利用极坐标代换，半径范围为 $0 \le r \le \frac{\sqrt{3}}{2} R$：
     $$S = \int_0^{2\pi} d\theta \int_0^{\frac{\sqrt{3}}{2} R} \frac{R}{\sqrt{R^2-r^2}} r dr = 2\pi R \left[ -\sqrt{R^2-r^2} \right]_0^{\frac{\sqrt{3}}{2} R}$$
     $$S = 2\pi R \left( -\sqrt{R^2 - \frac{3}{4} R^2} - (-R) \right) = 2\pi R \left( R - \frac{R}{2} \right) = \pi R^2$$

## 六、 备考解题套路与易错防范

### 1. 改变积分次序画图要精确
在交换积分次序时，务必在草稿纸上画出积分区域的准确边界线，并标清交点坐标。然后通过“穿线法”（水平穿线对应先对 $x$ 积分，竖直穿线对应先对 $y$ 积分）来定出新的积分上下限。

### 2. 重积分体积元的变量不要漏
* 极坐标下的 $r$；
* 柱坐标下的 $r$；
* 球坐标下的 $\rho^2 \sin\varphi$。
在代换时，漏掉这些因子是整张考卷中最常见、最痛心的失分点。建议在列出积分式的第一时间，就把体积元/面积元完整写下。
