# 工科数学分析期末大题突破：第 19 章 含参变量的积分（PPT例题与详解）

> **期末核心导读**：含参变量的积分是分析学中非常有特色且必考的计算章节。期末大题的考察重点非常明确：**含参常义积分的计算（核心技巧是引入参数对参数求导，再积分回去，或交换积分次序）**，**含参反常积分的一致收敛性判定（Weierstrass $M$-判别法和 Dirichlet/Abel 判别法）**，以及**利用 Beta 与 Gamma 函数计算复杂的反常积分（期末送分大题，公式必背）**。

---

## 一、 考情分析与考查频率

| 大题考点 | 常见考查形式 | 考查频率 |
| :--- | :--- | :---: |
| **利用 Beta/Gamma 函数计算积分** | 将形如 $\int_0^{\pi/2} \sin^m x \cos^n x dx$ 或 $\int_0^{+\infty} \frac{x^{a}}{(1+x)^b} dx$ 的积分化为 Euler 积分求解 | ★★★★★ |
| **积分号下求导法计算定积分** | 引入参数 $\theta$，将积分设为 $I(\theta)$，求导计算后，利用初值条件积分回去求出原积分值 | ★★★★★ |

## 二、 核心公式与 Euler 积分（必背）

### 1. 变限积分求导公式（Leibniz 公式）
设 $I(y) = \int_{\alpha(y)}^{\beta(y)} f(x,y) dx$，若 $f(x,y)$ 及其偏导数 $f_y$ 连续，且 $\alpha(y), \beta(y)$ 可导，则：
$$\frac{d}{dy} I(y) = f(\beta(y), y) \cdot \beta'(y) - f(\alpha(y), y) \cdot \alpha'(y) + \int_{\alpha(y)}^{\beta(y)} \frac{\partial f(x,y)}{\partial y} dx$$

### 2. Gamma 函数与 Beta 函数
* **Gamma 函数**：
  $$\Gamma(s) = \int_0^{+\infty} x^{s-1} e^{-x} dx \quad (s > 0)$$
  * **递推性质**：$\Gamma(s+1) = s \Gamma(s)$；当 $n$ 为正整数时，$\Gamma(n+1) = n!$。
  * **重要常数值**：$\Gamma(1) = 1, \quad \Gamma\left(\frac{1}{2}\right) = \sqrt{\pi}$。
* **Beta 函数**：
  $$B(p,q) = \int_0^1 x^{p-1} (1-x)^{q-1} dx \quad (p > 0, q > 0)$$
  * **对称性**：$B(p,q) = B(q,p)$。
  * **三角函数形式**：
    $$B(p,q) = 2 \int_0^{\pi/2} (\sin\theta)^{2p-1} (\cos\theta)^{2q-1} d\theta$$
  * **无穷区间形式**：
    $$B(p,q) = \int_0^{+\infty} \frac{t^{p-1}}{(1+t)^{p+q}} dt$$
* **Beta 与 Gamma 的连接桥梁（极高频计算核心）**：
  $$B(p,q) = \frac{\Gamma(p)\Gamma(q)}{\Gamma(p+q)}$$

## 三、 PPT例题详解：§1 含参变量常义积分

### 例 3.1（交换积分次序巧妙化简）
**【题目】** 计算 $I = \int_0^1 \frac{x^b - x^a}{\ln x} dx$ ($b > a > 0$)。
* **考点**：常义积分号下积分（交换积分次序）。
* **解析**：
  观察被积函数，注意到其分子可表示为：
  $$\frac{x^b - x^a}{\ln x} = \int_a^b x^y dy \quad (x \in (0, 1])$$
  所以原积分可写为：
  $$I = \int_0^1 \left( \int_a^b x^y dy \right) dx$$
  由于 $f(x,y) = x^y$ 在闭区域 $[0, 1] \times [a, b]$（因为 $a > 0$）上连续，可以应用常义二重积分的交换次序定理：
  $$I = \int_a^b \left( \int_0^1 x^y dx \right) dy$$
  我们先计算内层关于 $x$ 的积分：
  $$\int_0^1 x^y dx = \left[ \frac{x^{y+1}}{y+1} \right]_0^1 = \frac{1}{y+1}$$
  再计算外层关于 $y$ 的积分：
  $$I = \int_a^b \frac{dy}{y+1} = \left[ \ln(y+1) \right]_a^b = \ln \frac{b+1}{a+1}$$

### 例 4.1（积分号下求导典型大题）
**【题目】** 计算 $I(\theta) = \int_0^{\pi} \ln(1 + \theta \cos x) dx$ ($|\theta| < 1$)。
* **考点**：积分号下求导法，万能公式积分（期末超高频大题）。
* **解析**：
  1. **积分号下求导**：
     对 $\theta$ 求导数，并在积分号下求导：
     $$I'(\theta) = \int_0^{\pi} \frac{\cos x}{1 + \theta \cos x} dx = \frac{1}{\theta} \int_0^{\pi} \frac{\theta \cos x + 1 - 1}{1 + \theta \cos x} dx$$
     $$I'(\theta) = \frac{1}{\theta} \int_0^{\pi} \left( 1 - \frac{1}{1 + \theta \cos x} \right) dx = \frac{\pi}{\theta} - \frac{1}{\theta} \int_0^{\pi} \frac{dx}{1 + \theta \cos x}$$
  2. **计算分母积分**：
     设 $J = \int_0^{\pi} \frac{dx}{1 + \theta \cos x}$。作万能代换 $t = \tan \frac{x}{2}$，则 $\cos x = \frac{1-t^2}{1+t^2}, dx = \frac{2dt}{1+t^2}$：
     $$J = \int_0^{+\infty} \frac{\frac{2dt}{1+t^2}}{1 + \theta \frac{1-t^2}{1+t^2}} = \int_0^{+\infty} \frac{2dt}{(1+\theta) + (1-\theta)t^2}$$
     由于 $|\theta| < 1$，分母两项皆正。提取常数：
     $$J = \frac{2}{1-\theta} \int_0^{+\infty} \frac{dt}{t^2 + \left(\sqrt{\frac{1+\theta}{1-\theta}}\right)^2} = \frac{2}{1-\theta} \left[ \sqrt{\frac{1-\theta}{1+\theta}} \arctan\left( t \sqrt{\frac{1-\theta}{1+\theta}} \right) \right]_0^{+\infty}$$
     $$J = \frac{2}{\sqrt{1-\theta^2}} \cdot \frac{\pi}{2} = \frac{\pi}{\sqrt{1-\theta^2}}$$
  3. **求得导数表达式**：
     代回 $I'(\theta)$：
     $$I'(\theta) = \frac{\pi}{\theta} - \frac{\pi}{\theta \sqrt{1-\theta^2}} = \pi \left( \frac{\sqrt{1-\theta^2} - 1}{\theta \sqrt{1-\theta^2}} \right)$$
  4. **积分还原**：
     $$I(\theta) = \pi \int \left( \frac{1}{\theta} - \frac{1}{\theta \sqrt{1-\theta^2}} \right) d\theta$$
     利用代换 $\theta = \sin u$，可积得：
     $$I(\theta) = \pi \ln(1 + \sqrt{1-\theta^2}) + C$$
     代入初始条件 $I(0) = \int_0^{\pi} \ln 1 dx = 0$：
     $$0 = \pi \ln 2 + C \implies C = -\pi \ln 2$$
     故得结论：
     $$I(\theta) = \pi \ln \frac{1 + \sqrt{1-\theta^2}}{2}$$

## 四、 PPT例题详解：§2 含参变量反常积分

### 例 3.4（分部积分消去分母）
**【题目】** 计算反常积分 $I = \int_0^{+\infty} \frac{\cos ax - \cos bx}{x^2} dx$ ($b > a > 0$)。
* **考点**：含参反常积分的一致收敛性与分部积分法。
* **解析**：
  使用分部积分法以消去分母中的二次项：
  $$I = \int_0^{+\infty} (\cos ax - \cos bx) d\left( -\frac{1}{x} \right) = \left[ -\frac{\cos ax - \cos bx}{x} \right]_0^{+\infty} - \int_0^{+\infty} \left(-\frac{1}{x}\right) (-a \sin ax + b \sin bx) dx$$
  * 考察端点值：当 $x \to +\infty$ 时，分母趋于无穷，极限为 0；
    当 $x \to 0^+$ 时，利用等价无穷小 $\lim_{x\to 0^+} \frac{\cos ax - \cos bx}{x} = 0$。
  所以第一项（边界项）完全为 0。
  原式简化为：
  $$I = \int_0^{+\infty} \frac{b \sin bx - a \sin ax}{x} dx = b \int_0^{+\infty} \frac{\sin bx}{x} dx - a \int_0^{+\infty} \frac{\sin ax}{x} dx$$
  由于已知 Dirichlet 积分公式（与参数无关）：对任意常数 $k > 0$，有 $\int_0^{+\infty} \frac{\sin kx}{x} dx = \frac{\pi}{2}$。
  代入：
  $$I = b \left( \frac{\pi}{2} \right) - a \left( \frac{\pi}{2} \right) = \frac{\pi}{2} (b - a)$$

## 五、 PPT例题详解：§3 欧拉积分

### 例 2.1（指数衰减型化为 Gamma 函数）
**【题目】** 计算积分 $I = \int_0^{+\infty} x^{2n} e^{-x^2} dx$（$n$ 为正整数）。
* **考点**：指数复合项转化为标准 Gamma 积分。
* **解析**：
  令 $t = x^2 \implies x = t^{1/2}$。微分得 $dx = \frac{1}{2} t^{-1/2} dt$。代入得：
  $$I = \int_0^{+\infty} t^n e^{-t} \left( \frac{1}{2} t^{-1/2} dt \right) = \frac{1}{2} \int_0^{+\infty} t^{n - 1/2} e^{-t} dt$$
  根据 Gamma 函数的定义式：$\Gamma(s) = \int_0^{+\infty} t^{s-1} e^{-t} dt$，此处 $s - 1 = n - 1/2 \implies s = n + 1/2$：
  $$I = \frac{1}{2} \Gamma\left(n + \frac{1}{2}\right)$$
  利用递推关系：
  $$\Gamma\left(n + \frac{1}{2}\right) = \left(n - \frac{1}{2}\right) \left(n - \frac{3}{2}\right) \cdots \frac{1}{2} \Gamma\left(\frac{1}{2}\right) = \frac{(2n-1)!!}{2^n} \sqrt{\pi}$$
  所以：
  $$I = \frac{(2n-1)!!}{2^{n+1}} \sqrt{\pi}$$

---

### 例 3.2（根式型转化为 Beta 函数）
**【题目】** 计算 $\int_0^1 x^4 \sqrt{1-x^2} dx$。
* **考点**：根式积分的 Beta 函数转化。
* **解析**：
  令 $x^2 = t \implies x = t^{1/2}$，微分得 $dx = \frac{1}{2} t^{-1/2} dt$。
  将区间 $[0,1]$ 映射到 $[0,1]$：
  $$I = \int_0^1 t^2 (1-t)^{1/2} \left( \frac{1}{2} t^{-1/2} \right) dt = \frac{1}{2} \int_0^1 t^{3/2} (1-t)^{1/2} dt$$
  这符合 Beta 函数标准定义：$B(p,q) = \int_0^1 t^{p-1} (1-t)^{q-1} dt$，其中 $p-1 = 3/2 \implies p = 5/2$，$q-1 = 1/2 \implies q = 3/2$。
  $$I = \frac{1}{2} B\left(\frac{5}{2}, \frac{3}{2}\right)$$
  利用连接公式转化为 Gamma 函数计算：
  $$B\left(\frac{5}{2}, \frac{3}{2}\right) = \frac{\Gamma(5/2)\Gamma(3/2)}{\Gamma(4)}$$
  * $\Gamma(5/2) = \frac{3}{2} \cdot \frac{1}{2} \cdot \Gamma(1/2) = \frac{3}{4} \sqrt{\pi}$；
  * $\Gamma(3/2) = \frac{1}{2} \cdot \Gamma(1/2) = \frac{1}{2} \sqrt{\pi}$；
  * $\Gamma(4) = 3! = 6$。
  代入：
  $$B\left(\frac{5}{2}, \frac{3}{2}\right) = \frac{\frac{3}{8}\pi}{6} = \frac{\pi}{16}$$
  故原动积分值为：
  $$I = \frac{1}{2} \cdot \frac{\pi}{16} = \frac{\pi}{32}$$

### 例 3.4（分式形式转化为 Beta 函数）
**【题目】** 计算 $I = \int_0^{+\infty} \frac{\sqrt{x}}{(1+x)^2} dx$。
* **考点**：利用无穷区间形式的 Beta 函数求解。
* **解析**：
  利用 Beta 函数的无穷限定义式：
  $$B(p,q) = \int_0^{+\infty} \frac{x^{p-1}}{(1+x)^{p+q}} dx$$
  我们将被积函数的指数进行对齐：
  * 分子为 $x^{1/2} \implies p-1 = 1/2 \implies p = 3/2$；
  * 分母次数为 $2 \implies p+q = 2 \implies \frac{3}{2} + q = 2 \implies q = 1/2$。
  由于 $p = 3/2 > 0$ 且 $q = 1/2 > 0$，积分收敛，且其值等于：
  $$I = B\left(\frac{3}{2}, \frac{1}{2}\right) = \frac{\Gamma(3/2)\Gamma(1/2)}{\Gamma(2)}$$
  计算：
  * $\Gamma(3/2) = \frac{1}{2} \sqrt{\pi}$；
  * $\Gamma(1/2) = \sqrt{\pi}$；
  * $\Gamma(2) = 1! = 1$。
  代入得：
  $$I = \frac{\frac{1}{2}\sqrt{\pi} \cdot \sqrt{\pi}}{1} = \frac{\pi}{2}$$

## 六、 备考解题套路与易错防范

### 1. 区分 Gamma 函数的阶数差
记忆 Gamma 函数阶数时极易将 $\Gamma(n)$ 与 $n!$ 混淆。记住：
$$\Gamma(n) = (n-1)!$$
例如：$\Gamma(4) = 3! = 6$。多写错一个 1，最终答案会相差好几倍。

### 2. 一致收敛是换序的“入场券”
在解答反常积分求导、求极限或积分交换次序的大题时，**第一步必须先写出反常积分在参数区间上一致收敛的说明**（通常通过 Weierstrass $M$-判别法：$|f(x,y)| \le g(x)$ 且 $\int g dx$ 收敛）。直接换序而不加文字说明通常要被扣除大题的“步骤分”。
