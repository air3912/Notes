# 工科数学分析期末大题突破：第 18 章 极值与条件极值（PPT例题与详解）

> **期末核心导读**：本章是微分法最核心的实际应用章节，期末大题必考。主要题型分为两类：**无条件极值（寻找驻点，并利用 Hessian 矩阵判定）**与**条件极值（利用拉格朗日乘数法求解，通常与几何面积、体积或物理最值模型结合）**。掌握拉格朗日乘数法的代数消元技巧是拿满分大题的关键。

## 一、 考情分析与考查频率

| 大题考点 | 常见考查形式 | 考查频率 |
| :--- | :--- | :---: |
| **拉格朗日乘数法（单一/双重约束）** | 求目标函数在单约束或双约束（如两曲面交线）下的最值（几何应用、最值证明） | ★★★★★ |

| **最小二乘法直线拟合** | 给定几组数据点，通过正规方程组拟合直线 $y=ax+b$ | ★★☆☆☆ |

## 二、 核心方法与判定定理（必背）

### 1. 二元无条件极值判定（Hessian 判定法）
设 $(x_0, y_0)$ 是 $f(x,y)$ 的驻点（即 $f_x = 0, f_y = 0$）。记二阶偏导数值为：
$$A = f_{xx}(x_0, y_0), \quad B = f_{xy}(x_0, y_0), \dots, C = f_{yy}(x_0, y_0)$$
计算判别式 $H = AC - B^2$：
1. **若 $H > 0$**：存在极值。
   * 若 $A < 0$，则为**极大值**；
   * 若 $A > 0$，则为**极小值**。
2. **若 $H < 0$**：**不存在极值**（该点为鞍点）。
3. **若 $H = 0$**：**失效**，必须通过定义或路径符号进行特殊判定。

### 2. 拉格朗日乘数法（Lagrange Multipliers）
* **单一约束下**：求 $f(x,y,z)$ 在约束 $g(x,y,z) = 0$ 下的极值。
  构造拉格朗日函数 $L = f - \lambda g$，解偏导方程组：
  $$\begin{cases} f_x = \lambda g_x \\ f_y = \lambda g_y \\ f_z = \lambda g_z \\ g(x,y,z) = 0 \end{cases}$$
* **双重约束下**：求 $f(x,y,z)$ 在约束 $g(x,y,z) = 0$ 和 $h(x,y,z) = 0$ 下的极值。
  构造拉格朗日函数 $L = f - \lambda g - \mu h$，解偏导方程组：
  $$\nabla f = \lambda \nabla g + \mu \nabla h \quad \text{与} \quad g=0, h=0$$

## 三、 PPT例题详解：§1 极值与最值

### 例 1.1（无条件极值典型判定）
**【题目】** 求函数 $f(x,y) = xy(a-x-y)$ ($a \ne 0$) 的极值。
* **考点**：无条件极值判定步骤，Hessian 判别法。
* **解析**：
  1. **寻找驻点**：
     $$f_x = y(a-2x-y) = 0$$
     $$f_y = x(a-x-2y) = 0$$
     * 若 $x=0$，代回得 $y(a-y)=0 \implies (0,0), (0,a)$ 为驻点；
     * 若 $y=0$，代回得 $x(a-x)=0 \implies (0,0), (a,0)$ 为驻点；
     * 若 $x \ne 0, y \ne 0$，则：
       $$\begin{cases} 2x+y = a \\ x+2y = a \end{cases} \implies x = y = \frac{a}{3}$$
     综上，驻点为 $(0,0), (0,a), (a,0)$ 和 $\left(\frac{a}{3}, \frac{a}{3}\right)$。
  2. **计算二阶偏导数**：
     $$A = f_{xx} = -2y, \quad B = f_{xy} = a - 2x - 2y, \quad C = f_{yy} = -2x$$
  3. **利用 Hessian 判定各驻点**：
     * **对于 $(0,0), (0,a), (a,0)$**：
       代入可算得 $AC - B^2 = -a^2 < 0$。根据判定定理，这些点均为鞍点，**无极值**。
     * **对于 $\left(\frac{a}{3}, \frac{a}{3}\right)$**：
       $$A = -\frac{2a}{3}, \quad B = -\frac{a}{3}, \quad C = -\frac{2a}{3}$$
       $$H = AC - B^2 = \left(-\frac{2a}{3}\right)\left(-\frac{2a}{3}\right) - \left(-\frac{a}{3}\right)^2 = \frac{4a^2}{9} - \frac{a^2}{9} = \frac{a^2}{3} > 0$$
       因为 $H > 0$，此点必有极值。
       * 当 $a > 0$ 时，$A = -\frac{2a}{3} < 0$，函数取得**局部极大值**，极大值为 $f\left(\frac{a}{3}, \frac{a}{3}\right) = \frac{a^3}{27}$；
       * 当 $a < 0$ 时，$A = -\frac{2a}{3} > 0$，函数取得**局部极小值**，极小值为 $f\left(\frac{a}{3}, \frac{a}{3}\right) = \frac{a^3}{27}$。

### 例 1.2（Hessian 失效的判定反例）
**【题目】** 讨论 $f(x,y) = x^2 - 2xy^2$ 在 $(0,0)$ 处的极值。
* **考点**：$AC-B^2=0$ 判定失效时的符号分析法（极具启发性）。
* **解析**：
  1. **寻找驻点**：
     $$f_x = 2x - 2y^2 = 0 \implies x = y^2$$
     $$f_y = -4xy = 0 \implies -4y^3 = 0 \implies y = 0$$
     所以 $(0,0)$ 是唯一的驻点。
  2. **计算二阶偏导数**：
     $$f_{xx} = 2, \quad f_{xy} = -4y, \quad f_{yy} = -4x$$
     在 $(0,0)$ 处，有 $A = 2, B = 0, C = 0$。
     判别式 $H = AC - B^2 = 2 \cdot 0 - 0^2 = 0$。此时二阶判定法失效！
  3. **路径符号法分析**：
     注意到 $f(0,0) = 0$。我们将函数变形为：
     $$f(x,y) = x(x - 2y^2)$$
     在任意包含原点的邻域内：
     * 若沿抛物线 $x = y^2$ 逼近原点：
       $$f(y^2, y) = y^2(y^2 - 2y^2) = -y^4 < 0 \quad (y \ne 0)$$
     * 若沿抛物线 $x = 3y^2$ 逼近原点：
       $$f(3y^2, y) = 3y^2(3y^2 - 2y^2) = 3y^4 > 0 \quad (y \ne 0)$$
     由于在原点附近的任意邻域内，函数值既能大于 0 也能小于 0，因此 $f(0,0)$ 不是极值。

### 例 2.1（闭区域最值问题）
**【题目】** 在以 $O(0,0)$、$A(1,0)$ 和 $B(0,1)$ 为顶点的三角形闭区域上，求一点使其到三个顶点的距离平方和达到最大和最小，并求出其最值。
* **考点**：有界闭区域上最值求解步骤。
* **解析**：
  设所求点为 $P(x,y)$。目标函数为距离平方和：
  $$f(x,y) = (x-0)^2 + (y-0)^2 + (x-1)^2 + (y-0)^2 + (x-0)^2 + (y-1)^2$$
  $$f(x,y) = 3x^2 + 3y^2 - 2x - 2y + 2$$
  1. **探讨区域内部驻点**：
     $$\begin{cases} f_x = 6x - 2 = 0 \implies x = 1/3 \\ f_y = 6y - 2 = 0 \implies y = 1/3 \end{cases}$$
     点 $\left(\frac{1}{3}, \frac{1}{3}\right)$ 在三角形内部。此时函数值为：
     $$f\left(\frac{1}{3}, \frac{1}{3}\right) = 3\left(\frac{1}{9}\right) + 3\left(\frac{1}{9}\right) - \frac{2}{3} - \frac{2}{3} + 2 = \frac{4}{3}$$
  2. **探讨边界上的最值情况**：
     * **边界 $OA$**：$y = 0, 0 \le x \le 1$。
       $$f(x,0) = 3x^2 - 2x + 2$$
       在一元意义下，极小值在 $x = 1/3$ 取得，此时 $f(1/3, 0) = \frac{5}{3}$。
       端点值 $f(0,0) = 2$，$f(1,0) = 3$。
     * **边界 $OB$**：$x = 0, 0 \le y \le 1$。
       同理，极小值在 $y = 1/3$ 取得，此时 $f(0, 1/3) = \frac{5}{3}$。
       端点值 $f(0,1) = 3$。
     * **边界 $AB$**：$y = 1-x, 0 \le x \le 1$。
       $$f(x, 1-x) = 3x^2 + 3(1-x)^2 - 2x - 2(1-x) + 2 = 6x^2 - 6x + 3$$
       极小值在 $x = 1/2$ 取得，此时 $f(1/2, 1/2) = \frac{3}{2}$。
  3. **数值比较与结论**：
     比较上述所有候选点：
     * 内部驻点 $\left(\frac{1}{3}, \frac{1}{3}\right)$：$f = \frac{4}{3}$ （最小值）；
     * 边界极小值：$\frac{5}{3}$，$\frac{3}{2}$；
     * 顶点：$O(0,0) \implies 2$；$A(1,0) \implies 3$；$B(0,1) \implies 3$。
     * **结论**：距离平方和的最小值为 $\frac{4}{3}$，在点 $\left(\frac{1}{3}, \frac{1}{3}\right)$（即三角形的重心）处取得；最大值为 $3$，在顶点 $A(1,0)$ 和 $B(0,1)$ 处取得。

## 四、 PPT例题详解：§2 拉格朗日乘数法

### 例 1.2（双重约束条件下的几何极值大题）
**【题目】** 求平面 $x + y + z = 0$ 与椭球面 $x^2 + y^2 + 4z^2 = 1$ 相交而成的截面椭圆的面积。
* **考点**：双重约束拉格朗日乘数法，空间解析几何与极值的结合（极高频期末压轴大题）。
* **解析**：
  相交轨迹是一个中心在原点的椭圆。椭圆的面积公式为 $S = \pi a b$，其中 semi-major 和 semi-minor 轴 $a, b$ 分别对应于原点到该椭圆上点距离的最大值和最小值。
  因此，问题转化为：在约束 $x+y+z=0$ 和 $x^2+y^2+4z^2=1$ 下，求距离平方函数 $f(x,y,z) = x^2+y^2+z^2$ 的最值。
  1. **构造拉格朗日函数**：
     $$L = x^2+y^2+z^2 - \lambda (x^2+y^2+4z^2-1) - 2\mu (x+y+z)$$
  2. **列出偏导方程组**：
     $$\begin{cases} L_x = 2x - 2\lambda x - 2\mu = 0 \implies x(1-\lambda) = \mu \quad \text{--- (1)} \\ L_y = 2y - 2\lambda y - 2\mu = 0 \implies y(1-\lambda) = \mu \quad \text{--- (2)} \\ L_z = 2z - 8\lambda z - 2\mu = 0 \implies z(1-4\lambda) = \mu \quad \text{--- (3)} \\ x+y+z = 0 \quad \text{--- (4)} \\ x^2+y^2+4z^2 = 1 \quad \text{--- (5)} \end{cases}$$
  3. **代数消元求解**：
     由 (1) 和 (2) 得：
     $$(x-y)(1-\lambda) = 0$$
     这引导我们分两种情况讨论：
     * **情况一：$x \ne y \implies \lambda = 1$**：
       代入 (1) 得 $\mu = 0$。将 $\mu = 0$ 代入 (3) 得：
       $$z(1-4) = 0 \implies z = 0$$
       代入平面方程 (4) 得：
       $$x + y = 0 \implies y = -x$$
       代回椭球面方程 (5)：
       $$x^2 + (-x)^2 + 4(0)^2 = 1 \implies 2x^2 = 1 \implies x^2 = y^2 = \frac{1}{2}$$
       此时的距离平方为：
       $$d_1^2 = x^2+y^2+z^2 = \frac{1}{2} + \frac{1}{2} + 0 = 1 \implies d_1 = 1$$
     * **情况二：$x = y$**：
       代入平面方程 (4) 得：
       $$2x + z = 0 \implies z = -2x$$
       代回椭球面方程 (5)：
       $$x^2 + x^2 + 4(-2x)^2 = 1 \implies 18x^2 = 1 \implies x^2 = y^2 = \frac{1}{18}$$
       此时 $z^2 = 4x^2 = \frac{4}{18}$。距离平方为：
       $$d_2^2 = x^2+y^2+z^2 = \frac{1}{18} + \frac{1}{18} + \frac{4}{18} = \frac{6}{18} = \frac{1}{3} \implies d_2 = \frac{1}{\sqrt{3}}$$
  4. **求得面积**：
     根据物理/几何意义，最大距离即为长半轴 $a = 1$，最小距离即为短半轴 $b = \frac{1}{\sqrt{3}}$。
     截面椭圆的面积为：
     $$S = \pi a b = \pi \cdot 1 \cdot \frac{1}{\sqrt{3}} = \frac{\pi}{\sqrt{3}}$$

### 例 2.2（带幂次的和约束）
**【题目】** 求 $x>0, y>0, z>0$ 时，函数 $f(x,y,z) = \ln x + \ln y + 2\ln z$ 在球面 $x^2 + y^2 + z^2 = 4R^2$ 上的最大值。
* **考点**：单约束条件极值。
* **解析**：
  1. **构造拉格朗日函数**：
     $$L = \ln x + \ln y + 2\ln z - \lambda (x^2+y^2+z^2-4R^2)$$
  2. **列偏导方程组**：
     $$\begin{cases} L_x = \frac{1}{x} - 2\lambda x = 0 \implies 2\lambda x^2 = 1 \\ L_y = \frac{1}{y} - 2\lambda y = 0 \implies 2\lambda y^2 = 1 \\ L_z = \frac{2}{z} - 2\lambda z = 0 \implies \lambda z^2 = 1 \implies 2\lambda z^2 = 2 \\ x^2+y^2+z^2 = 4R^2 \end{cases}$$
  3. **求和代入**：
     将前三式相加：
     $$2\lambda x^2 + 2\lambda y^2 + 2\lambda z^2 = 1 + 1 + 2 = 4 \implies 2\lambda (x^2+y^2+z^2) = 4$$
     代入球面方程 $x^2+y^2+z^2 = 4R^2$：
     $$2\lambda (4R^2) = 4 \implies 2\lambda = \frac{1}{R^2}$$
  4. **解出坐标与最值**：
     代回原方程组：
     * $x^2 = \frac{1}{2\lambda} = R^2 \implies x = R$（因为 $x>0$）；
     * $y^2 = \frac{1}{2\lambda} = R^2 \implies y = R$；
     * $z^2 = \frac{2}{2\lambda} = 2R^2 \implies z = \sqrt{2}R$。
     唯一稳定点的函数值为：
     $$f(R, R, \sqrt{2}R) = \ln R + \ln R + 2\ln(\sqrt{2}R) = 2\ln R + \ln(2R^2) = 4\ln R + \ln 2$$
     * **结论**：最大值为 $4\ln R + \ln 2$。

## 五、 备考解题套路与易错防范

### 1. 拉格朗日法解方程的绝招：相加/相比法
拉格朗日方程组（如 $\frac{1}{x} = 2\lambda x, \frac{2}{z} = 2\lambda z$）往往是非线性的，很多学生卡在求解这一步。
* **招数一：两两相比**。消去 $\lambda$，如 $\frac{1/x}{1/y} = \frac{2\lambda x}{2\lambda y} \implies \frac{y}{x} = \frac{x}{y} \implies x^2 = y^2$。
* **招数二：乘变量再相加**。如例 2.2 中，将各式分别乘以对应自变量，化为 $2\lambda x^2 = 1$, $2\lambda z^2 = 2$，相加利用约束条件直接锁死 $\lambda$ 的值。

### 2. 区分局部极值与最值
在应用题（如体积最大、引力最小）中，若求得的拉格朗日稳定点是**唯一的**，且实际问题从物理几何上明显存在最值，可以直接断定该稳定点即为所求的最值点，不需要再使用复杂的二阶判定。
