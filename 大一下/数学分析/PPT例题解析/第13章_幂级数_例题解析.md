# 工科数学分析期末大题突破：第 13 章 幂级数（PPT例题与详解）

> **期末核心导读**：幂级数是期末考试大题中分值极高、解题套路最为成熟的章节。工科数分最常考的题型包括：**求幂级数的收敛域（必考且端点必单独讨论）**，**函数展开为幂级数（常考指定中心 $x_0$ 展开、部分分式展开与求导积分展开）**，以及**幂级数求和函数（通过逐项求导或积分还原为几何级数等母式）**。

---

## 二、 考情分析与考查频率

| 大题考点 | 常见考查形式 | 考查频率 |
| :--- | :--- | :---: |
| **求收敛域（包含端点判定）** | 结合比值法/根值法求收敛半径 $R$，再代入端点使用数项级数（如交错级数、比较法）讨论 | ★★★★★ |
| **幂级数求和函数** | 给定级数，通过逐项积分（消去分母 $n$）或逐项求导（消去分子 $n$）法求出和函数并指出其定义域 | ★★★★★ |
| **初等函数幂级数展开** | 将函数展开为 $x$ 或 $(x-x_0)$ 的级数。有理式多用部分分式拆分，三角/对数/反三角用逐项积分或求导 | ★★★★★ |
| **数值计算与误差估计** | 使用泰勒展开计算定积分（如无法初等积分的 $\int_0^1 e^{-x^2} dx$）并在给定精度下截断估算项数 | ★★★☆☆ |

## 二、 常用初等函数麦克劳林展开式（必背母式）

| 函数 $f(x)$ | 幂级数展开式 | 收敛区间 |
| :--- | :--- | :---: |
| **$\frac{1}{1-x}$** | $\sum_{n=0}^{\infty} x^n = 1 + x + x^2 + \dots$ | $(-1, 1)$ |
| **$e^x$** | $\sum_{n=0}^{\infty} \frac{x^n}{n!} = 1 + x + \frac{x^2}{2!} + \dots$ | $(-\infty, +\infty)$ |
| **$\sin x$** | $\sum_{n=0}^{\infty} (-1)^n \frac{x^{2n+1}}{(2n+1)!} = x - \frac{x^3}{3!} + \dots$ | $(-\infty, +\infty)$ |
| **$\cos x$** | $\sum_{n=0}^{\infty} (-1)^n \frac{x^{2n}}{(2n)!} = 1 - \frac{x^2}{2!} + \dots$ | $(-\infty, +\infty)$ |
| **$\ln(1+x)$** | $\sum_{n=1}^{\infty} (-1)^{n-1} \frac{x^n}{n} = x - \frac{x^2}{2} + \frac{x^3}{3} - \dots$ | $[-1, 1)$ |
| **$\arctan x$** | $\sum_{n=0}^{\infty} (-1)^n \frac{x^{2n+1}}{2n+1} = x - \frac{x^3}{3} + \frac{x^5}{5} - \dots$ | $[-1, 1]$ |

## 三、 PPT例题详解：§1 收敛半径与收敛域

### 例 2.1（三组基本收敛域的对照分析）
**【题目】** 讨论下列幂级数的收敛半径与收敛域：
1. $\sum_{n=1}^{\infty} \frac{x^n}{n}$
2. $\sum_{n=1}^{\infty} \frac{x^n}{n^2}$
3. $\sum_{n=1}^{\infty} n x^n$
* **考点**：收敛半径相同但边界收敛行为不同的经典展示。
* **解析**：
  三组级数的收敛半径计算如下：
  1. $a_n = \frac{1}{n} \implies R = \lim_{n\to\infty} \left|\frac{a_n}{a_{n+1}}\right| = \lim_{n\to\infty} \frac{n+1}{n} = 1$。
     * 端点 $x = 1$：级数为 $\sum \frac{1}{n}$（调和级数，发散）；
     * 端点 $x = -1$：级数为 $\sum \frac{(-1)^n}{n}$（交错级数，由莱布尼茨定理，收敛）。
     * **收敛域为 $[-1, 1)$**。
  2. $a_n = \frac{1}{n^2} \implies R = 1$。
     * 端点 $x = \pm 1$：级数为 $\sum \frac{1}{n^2}$ 和 $\sum \frac{(-1)^n}{n^2}$，由于 $p=2>1$，两端点均绝对收敛。
     * **收敛域为 $[-1, 1]$**。
  3. $a_n = n \implies R = \lim_{n\to\infty} \frac{n}{n+1} = 1$。
     * 端点 $x = \pm 1$：级数为 $\sum (\pm 1)^n n$，通项不趋于 0，必定发散。
     * **收敛域为 $(-1, 1)$**。

### 例 2.2（根值法与上极限）
**【题目】** 求级数 $\sum_{n=1}^{\infty} [2 + (-1)^n]^n x^n$ 的收敛半径。
* **考点**：根值法（Cauchy-Hadamard 定理）与上极限（$\limsup$）。
* **解析**：
  这里通项系数为 $a_n = [2 + (-1)^n]^n$。
  我们求上极限：
  $$\limsup_{n\to\infty} \sqrt[n]{|a_n|} = \limsup_{n\to\infty} [2 + (-1)^n]$$
  由于序列的值在 1 和 3 之间交替变化：
  * 当 $n$ 为偶数时，$2 + (-1)^n = 3$；
  * 当 $n$ 为奇数时，$2 + (-1)^n = 1$。
  所以其上极限（最大聚点）为 3。
  根据收敛半径公式：
  $$R = \frac{1}{\limsup_{n\to\infty} \sqrt[n]{|a_n|}} = \frac{1}{3}$$
  * **避错提示**：大题中如果通项含 $(-1)^n$ 且影响系数大小，必须使用上极限。

### 例 2.3（平移中心幂级数的收敛域）
**【题目】** 求幂级数 $\sum_{n=1}^{\infty} \frac{(x-2)^n}{2^n n}$ 的收敛域。
* **考点**：收敛域的平移与比值法判定。
* **解析**：
  设 $t = x-2$，级数化为 $\sum_{n=1}^{\infty} \frac{t^n}{2^n n}$。
  利用比值法求收敛半径：
  $$R_t = \lim_{n\to\infty} \left| \frac{a_n}{a_{n+1}} \right| = \lim_{n\to\infty} \frac{2^{n+1}(n+1)}{2^n n} = 2$$
  所以对于 $t$，收敛区间为 $-2 < t < 2 \implies -2 < x-2 < 2 \implies 0 < x < 4$。
  现在单独讨论端点：
  * 当 $x = 4$ 时（即 $t = 2$）：级数为 $\sum_{n=1}^{\infty} \frac{2^n}{2^n n} = \sum_{n=1}^{\infty} \frac{1}{n}$（调和级数，发散）。
  * 当 $x = 0$ 时（即 $t = -2$）：级数为 $\sum_{n=1}^{\infty} \frac{(-2)^n}{2^n n} = \sum_{n=1}^{\infty} \frac{(-1)^n}{n}$（收敛）。
  * **收敛域为 $[0, 4)$**。

### 例 3.1（Stirling 公式在端点判定中的高阶应用）
**【题目】** 讨论 $\sum_{n=1}^{\infty} \frac{n!}{n^n} x^n$ 的收敛域。
* **考点**：比值法与 Stirling 公式确定端点敛散性（压轴大题高频考点）。
* **解析**：
  1. **求收敛半径**：
     $$R = \lim_{n\to\infty} \frac{a_n}{a_{n+1}} = \lim_{n\to\infty} \frac{n!}{n^n} \frac{(n+1)^{n+1}}{(n+1)!} = \lim_{n\to\infty} \frac{(n+1)^n}{n^n} = \lim_{n\to\infty} \left(1 + \frac{1}{n}\right)^n = e$$
     所以收敛半径为 $e$。开区间为 $(-e, e)$。
  2. **讨论端点 $x = \pm e$**：
     对于 $x = e$，级数为 $\sum_{n=1}^{\infty} \frac{n! e^n}{n^n}$。
     使用 **Stirling 公式**：$n! \sim \sqrt{2\pi n} \left(\frac{n}{e}\right)^n$。
     我们代入通项进行等价分析：
     $$u_n = \frac{n! e^n}{n^n} \sim \frac{\sqrt{2\pi n} \left(\frac{n}{e}\right)^n e^n}{n^n} = \sqrt{2\pi n} \to +\infty \quad (n\to\infty)$$
     因为通项 $u_n$ 当 $n\to\infty$ 时不趋于 0（实际上发散至 $+\infty$），所以根据发散必要性，级数在 $x = e$ 处发散。
     同理，在 $x = -e$ 处，通项为 $(-1)^n \frac{n! e^n}{n^n}$，其绝对值趋于 $+\infty$，通项同样不趋于 0，级数发散。
  * **结论**：原级数的收敛域为 **$(-e, e)$**。

## 四、 PPT例题详解：§2 幂级数的分析性质

### 例 2.3（用求导证明常数性质）
**【题目】** 设 $S(x) = \sum_{n=0}^{\infty} \frac{x^n}{n!}$，证明 $S(x) = e^x$。
* **考点**：逐项求导性质与常数变易法的混合证明。
* **证明**：
  因为级数在整个实数轴上收敛，所以可以在内部任意点进行逐项求导：
  $$S'(x) = \sum_{n=1}^{\infty} \frac{n x^{n-1}}{n!} = \sum_{n=1}^{\infty} \frac{x^{n-1}}{(n-1)!} = \sum_{k=0}^{\infty} \frac{x^k}{k!} = S(x)$$
  两边乘以 $e^{-x}$，得：
  $$e^{-x} (S'(x) - S(x)) = 0 \implies \left( e^{-x} S(x) \right)' = 0$$
  因此 $e^{-x} S(x) = C$（常数）。
  代入 $x=0$，有 $S(0) = \sum_{n=0}^{\infty} \frac{0^n}{n!} = 1$，且 $e^0 = 1$。
  得 $C = 1$，即 $e^{-x} S(x) = 1 \implies S(x) = e^x$。

---

### 例 2.4（逐项求导求和函数）
**【题目】** 求级数 $\sum_{n=0}^{\infty} \frac{x^{2n+1}}{2n+1}$ 的和函数。
* **考点**：逐项求导还原法。
* **解析**：
  易求该级数的收敛半径 $R = 1$。设和函数为 $S(x)$，定义域为 $[-1, 1]$（端点可由交错级数判定）。
  对于 $x \in (-1, 1)$，进行逐项求导：
  $$S'(x) = \left( \sum_{n=0}^{\infty} \frac{x^{2n+1}}{2n+1} \right)' = \sum_{n=0}^{\infty} x^{2n} = \frac{1}{1-x^2}$$
  由于 $S(0) = 0$，将导数积分回原函数：
  $$S(x) = \int_0^x \frac{dt}{1-t^2} = \frac{1}{2} \ln \frac{1+x}{1-x} \quad (x \in (-1, 1))$$
  对于端点 $x = \pm 1$：
  根据阿贝尔第二定理，由于级数在 $x = \pm 1$ 收敛，和函数在端点处应连续。
  然而 $\lim_{x\to 1^-} \frac{1}{2}\ln\frac{1+x}{1-x} = +\infty$ 并不收敛到有限值，
  * **更正说明**：原级数在 $x = \pm 1$ 实际上是**发散**的（因为级数在 $x=1$ 为 $\sum \frac{1}{2n+1}$ 发散；在 $x=-1$ 为 $\sum \frac{-1}{2n+1}$ 发散）。
  * **结论**：级数的和函数为 $S(x) = \frac{1}{2} \ln \frac{1+x}{1-x}$，收敛域为 **$(-1, 1)$**。

---

### 例 2.5（带平移的逐项求导）
**【题目】** 求级数 $\sum_{n=1}^{\infty} \frac{x^n}{n 2^n}$ 的和函数及收敛域。
* **考点**：逐项求导还原与收敛域判定。
* **解析**：
  比值法易得收敛半径 $R = 2$。
  设和函数为 $S(x)$，对 $x \in (-2, 2)$ 两边求导：
  $$S'(x) = \sum_{n=1}^{\infty} \frac{n x^{n-1}}{n 2^n} = \sum_{n=1}^{\infty} \frac{x^{n-1}}{2^n} = \frac{1}{2} \sum_{n=1}^{\infty} \left(\frac{x}{2}\right)^{n-1} = \frac{1}{2} \frac{1}{1 - x/2} = \frac{1}{2-x}$$
  因为 $S(0) = 0$，故：
  $$S(x) = \int_0^x \frac{dt}{2-t} = -\ln(2-x) + \ln 2 = \ln\frac{2}{2-x}$$
  讨论端点：
  * 当 $x = -2$ 时，原级数为 $\sum \frac{(-1)^n}{n}$，收敛；
  * 当 $x = 2$ 时，原级数为 $\sum \frac{1}{n}$，发散。
  根据阿贝尔第二定理，在收敛端点 $x=-2$ 处，和函数值可以通过极限取得：$S(-2) = \lim_{x\to -2^+} \ln\frac{2}{2-x} = \ln \frac{1}{2} = -\ln 2$。
  * **结论**：和函数为 $S(x) = \ln\frac{2}{2-x}$，收敛域为 **$[-2, 2)$**。

## 五、 PPT例题详解：§3 函数的幂级数展开

### 例 3.3（有理分式部分分式拆分展开）
**【题目】** 求 $f(x) = \frac{3}{3 + 5x - 2x^2}$ 在 $x=0$ 处的幂级数展开。
* **考点**：有理分式拆分与几何级数母式套用。
* **解析**：
  先因式分解分母：
  $$3 + 5x - 2x^2 = (3 - x)(1 + 2x)$$
  用待定系数法对有理式做部分分式分解：
  $$\frac{3}{(3-x)(1+2x)} = \frac{A}{3-x} + \frac{B}{1+2x} \implies 3 = A(1+2x) + B(3-x)$$
  * 令 $x = 3 \implies 7A = 3 \implies A = \frac{3}{7}$；
  * 令 $x = -\frac{1}{2} \implies \frac{7}{2} B = 3 \implies B = \frac{6}{7}$。
  代入得：
  $$f(x) = \frac{3}{7} \frac{1}{3-x} + \frac{6}{7} \frac{1}{1+2x} = \frac{1}{7} \frac{1}{1 - \frac{x}{3}} + \frac{6}{7} \frac{1}{1+2x}$$
  分别套用几何级数展开母式：
  * $\frac{1}{1 - \frac{x}{3}} = \sum_{n=0}^{\infty} \left( \frac{x}{3} \right)^n = \sum_{n=0}^{\infty} \frac{1}{3^n} x^n \quad (\text{收敛域 } |x| < 3)$；
  * $\frac{1}{1+2x} = \sum_{n=0}^{\infty} (-2x)^n = \sum_{n=0}^{\infty} (-1)^n 2^n x^n \quad (\text{收敛域 } |x| < \frac{1}{2})$。
  我们取收敛半径的交集：$R = \min(3, 1/2) = 1/2$。
  将级数合并：
  $$f(x) = \sum_{n=0}^{\infty} \left[ \frac{1}{7 \cdot 3^n} + \frac{6 \cdot (-1)^n 2^n}{7} \right] x^n \quad \left(x \in \left(-\frac{1}{2}, \frac{1}{2}\right)\right)$$

### 例 3.5（导数展开再积分还原）
**【题目】** 求 $f(x) = (1+x) \ln(1+x) - x$ 在 $x=0$ 处的展开式。
* **考点**：先求导展开，再逐项积分回原函数（高频题型）。
* **解析**：
  1. 对 $f(x)$ 求导数：
     $$f'(x) = \ln(1+x) + (1+x) \frac{1}{1+x} - 1 = \ln(1+x)$$
  2. 套用基本对数展开母式：
     $$f'(x) = \sum_{n=1}^{\infty} (-1)^{n-1} \frac{x^n}{n} \quad (x \in (-1, 1])$$
  3. 积分回去（利用 $f(0) = 0$）：
     $$f(x) = \int_0^x f'(t) dt = \int_0^x \left( \sum_{n=1}^{\infty} (-1)^{n-1} \frac{t^n}{n} \right) dt = \sum_{n=1}^{\infty} \frac{(-1)^{n-1}}{n} \int_0^x t^n dt$$
     $$f(x) = \sum_{n=1}^{\infty} \frac{(-1)^{n-1} x^{n+1}}{n(n+1)}$$
     为了让通项更简洁，可平移求和指标（令 $k = n+1$）：
     $$f(x) = \sum_{k=2}^{\infty} \frac{(-1)^k x^k}{k(k-1)}$$
     * **收敛性说明**：积分半径不变仍为 1。单独带入 $x = \pm 1$：由于分母为 $k^2$ 级别，两端点均收敛。
     * **结论**：展开式为 $\sum_{n=2}^{\infty} \frac{(-1)^n x^n}{n(n-1)}$，收敛域为 **$[-1, 1]$**。

---

### 例 3.6（指定非零中心展开）
**【题目】** 将 $f(x) = \ln(3-x)$ 在 $x_0 = 1$ 处展开。
* **考点**：指定中心 $x_0 \ne 0$ 的平移展开。
* **解析**：
  我们需要将函数表为 $(x-1)$ 的幂级数。令 $t = x-1 \implies x = t+1$。
  $$f(x) = \ln(3 - (t+1)) = \ln(2-t)$$
  为套用 $\ln(1+u)$ 母式，提出常数 2：
  $$\ln(2-t) = \ln\left[ 2 \left(1 - \frac{t}{2}\right) \right] = \ln 2 + \ln\left(1 - \frac{t}{2}\right)$$
  代入基本公式 $\ln(1+u) = \sum_{n=1}^{\infty} \frac{(-1)^{n-1} u^n}{n}$，此处 $u = -\frac{t}{2}$：
  $$\ln\left(1 - \frac{t}{2}\right) = \sum_{n=1}^{\infty} \frac{(-1)^{n-1} (-t/2)^n}{n} = -\sum_{n=1}^{\infty} \frac{t^n}{n 2^n}$$
  将 $t = x-1$ 代回：
  $$f(x) = \ln 2 - \sum_{n=1}^{\infty} \frac{(x-1)^n}{n 2^n}$$
  确定收敛区间：
  基本展开式需要 $\left|-\frac{t}{2}\right| < 1 \implies |t| < 2 \implies |x-1| < 2 \implies -1 < x < 3$。
  考查端点：
  * 当 $x = -1$ 时，级数为 $\sum \frac{(-2)^n}{n 2^n} = \sum \frac{(-1)^n}{n}$，收敛；
  * 当 $x = 3$ 时，级数为 $\sum \frac{2^n}{n 2^n} = \sum \frac{1}{n}$，发散。
  * **结论**：收敛域为 **$[-1, 3)$**。

### 例 3.7（幂级数的 Cauchy 乘积）
**【题目】** 求 $e^x \sin x$ 在 $x=0$ 处的幂级数展开（写出前几项，并给出通项）。
* **考点**：级数相乘（Cauchy 乘积）与复数辅助法。
* **解析**：
  **方法一（直接复数法，极为推荐）**：
  利用欧拉公式，我们知道 $e^x \sin x = \text{Im}\left( e^{(1+i)x} \right)$。
  展开复指数：
  $$e^{(1+i)x} = \sum_{n=0}^{\infty} \frac{(1+i)^n x^n}{n!}$$
  由于复数 $1+i = \sqrt{2} e^{i\frac{\pi}{4}}$，其 $n$ 次幂为 $(1+i)^n = 2^{\frac{n}{2}} e^{i\frac{n\pi}{4}}$。
  带回并取虚部：
  $$\text{Im}\left( (1+i)^n \right) = 2^{\frac{n}{2}} \sin \frac{n\pi}{4}$$
  所以展开式的通项为：
  $$e^x \sin x = \sum_{n=1}^{\infty} \frac{2^{\frac{n}{2}} \sin \frac{n\pi}{4}}{n!} x^n$$
  我们写出前几项：
  * $n=1 \implies \frac{\sqrt{2} \sin(\pi/4)}{1!} x = x$；
  * $n=2 \implies \frac{2 \sin(\pi/2)}{2!} x^2 = x^2$；
  * $n=3 \implies \frac{2\sqrt{2} \sin(3\pi/4)}{3!} x^3 = \frac{1}{3} x^3$；
  * $n=4 \implies \frac{4 \sin(\pi)}{4!} x^4 = 0$；
  * $n=5 \implies \frac{4\sqrt{2} \sin(5\pi/4)}{5!} x^5 = -\frac{1}{30} x^5$。
  即 $e^x \sin x = x + x^2 + \frac{x^3}{3} - \frac{x^5}{30} + \dots$

### 例 3.10（用级数求积分的数值近似）
**【题目】** 计算积分 $I = \int_0^1 e^{-x^2} dx$，要求精确到 $0.0001$。
* **考点**：无法初等表示的积分的幂级数求解与误差截断估计。
* **解析**：
  1. **将被积函数展开**：
     $$e^u = \sum_{n=0}^{\infty} \frac{u^n}{n!} \implies e^{-x^2} = \sum_{n=0}^{\infty} \frac{(-1)^n x^{2n}}{n!}$$
     该展开在整个 $\mathbb{R}$ 上一致收敛。
  2. **逐项积分**：
     $$I = \int_0^1 \left( \sum_{n=0}^{\infty} \frac{(-1)^n x^{2n}}{n!} \right) dx = \sum_{n=0}^{\infty} \frac{(-1)^n}{n!} \int_0^1 x^{2n} dx = \sum_{n=0}^{\infty} \frac{(-1)^n}{(2n+1) n!}$$
  3. **截断误差估计**：
     这是一个交错级数 $I = \sum_{n=0}^{\infty} (-1)^n b_n$，其中 $b_n = \frac{1}{(2n+1) n!}$ 递减趋于 0。
     对于交错级数，截断误差满足：
     $$|R_N| \le b_{N+1} = \frac{1}{(2N+3)(N+1)!}$$
     我们需要精确到 $0.0001 = 10^{-4}$，即要求 $|R_N| < 10^{-4}$：
     * 若 $N = 4 \implies b_5 = \frac{1}{11 \cdot 5!} = \frac{1}{11 \cdot 120} = \frac{1}{1320} \approx 0.00075$（精度不足）；
     * 若 $N = 5 \implies b_6 = \frac{1}{13 \cdot 6!} = \frac{1}{13 \cdot 720} = \frac{1}{9360} \approx 0.000107$（精度接近）；
     * 若 $N = 6 \implies b_7 = \frac{1}{15 \cdot 7!} = \frac{1}{15 \cdot 5040} = \frac{1}{75600} \approx 0.000013 < 10^{-4}$。
     所以取 $N = 6$（即前 7 项之和）可以保证误差小于 $0.0001$：
     $$I \approx \sum_{n=0}^{6} \frac{(-1)^n}{(2n+1)n!} = 1 - \frac{1}{3} + \frac{1}{10} - \frac{1}{42} + \frac{1}{216} - \frac{1}{1320} + \frac{1}{9360}$$
     计算可得数值。

## 六、 备考解题套路与易错防范

### 1. 求和函数的两步法
* **第一步：消除阻碍项**
  如果级数中有 $n$（如 $\sum n x^n$），就尝试**求导**或**积分**，将各项系数转化为几何级数形式。
* **第二步：平移指标对齐首项**
  求出导数或积分的和函数后，必须根据原级数的首项（例如 $n=0$ 还是 $n=1$）和自变量的幂次，通过积分常数 $C$ 进行精确的校正。

### 2. 级数展开大题的“安全区域”
大题中将函数展开，写出级数表达式后，**必须写出收敛区间或收敛域**，这是阅卷打分必看的要点。漏掉收敛范围一般扣除 2-3 分。
