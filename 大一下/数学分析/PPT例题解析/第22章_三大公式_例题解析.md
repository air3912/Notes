## 二、 核心公式与定理（必背）

### 1. 格林公式（Green's Formula）
设 $D$ 是平面单连通区域，其正向边界为 $\partial D$。若 $P(x,y), Q(x,y)$ 连续可微，则：
$$\oint_{\partial D} P dx + Q dy = \iint_D \left( \frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y} \right) dxdy$$

### 2. 高斯公式（Gauss's Formula）
设 $\Omega$ 是空间有界闭区域，其外侧边界为 $\Sigma$。若向量场 $\mathbf{F} = (P, Q, R)$ 连续可微，则：
$$\iint_{\Sigma} P dydz + Q dzdx + R dxdy = \iiint_{\Omega} \left( \frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z} \right) dV$$

### 3. 斯托克斯公式（Stokes's Formula）
设 $\Sigma$ 为有向曲面，其正向边界为 $\Gamma$（符合右手法则）。若 $\mathbf{F} = (P, Q, R)$ 连续可微，则：
$$\oint_{\Gamma} P dx + Q dy + R dz = \iint_{\Sigma} \text{curl} \mathbf{F} \cdot \mathbf{n} dS$$
其中旋度向量为：
$$\text{curl} \mathbf{F} = \begin{vmatrix} \mathbf{i} & \mathbf{j} & \mathbf{k} \\ \frac{\partial}{\partial x} & \frac{\partial}{\partial y} & \frac{\partial}{\partial z} \\ P & Q & R \end{vmatrix} = \left( \frac{\partial R}{\partial y} - \frac{\partial Q}{\partial z}, \frac{\partial P}{\partial z} - \frac{\partial R}{\partial x}, \frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y} \right)$$

## 三、 PPT例题详解：格林公式及路径无关

### 例 1.3（格林公式补线法大题模板）
**【题目】** 计算积分 $\int_L (e^x \sin y - my) dx + (e^x \cos y - m) dy$，其中 $L$ 为半圆周 $(x-a)^2 + y^2 = a^2$ ($y \ge 0, a > 0$) 的上半圆周部分，方向为从点 $A(2a, 0)$ 到原点 $O(0, 0)$。
* **考点**：非闭合曲线积分，格林公式的“补线法”。
* **解析**：
  1. **补线闭合区域**：
     曲线 $L$ 是从 $A(2a,0)$ 到 $O(0,0)$ 的上半圆弧（方向为顺时针）。它不是闭合的，无法直接用格林公式。
     我们添加一条沿 $x$ 轴的直线段 $L_1$：从 $O(0,0)$ 到 $A(2a,0)$。
     此时 $L + L_1$ 组成了一条闭合曲线，其方向为顺时针（负向）。其围成的区域 $D$ 是上半圆面。
  2. **计算在补线 $L_1$ 上的积分**：
     在直线段 $L_1$ 上：$y = 0 \implies dy = 0$。
     代入被积式：
     $$\int_{L_1} P dx + Q dy = \int_0^{2a} (e^x \sin 0 - m \cdot 0) dx = 0$$
     所以：
     $$\int_L = \oint_{L+L_1} - \int_{L_1} = \oint_{L+L_1} - 0 = \oint_{L+L_1}$$
  3. **利用格林公式计算闭曲线积分**：
     因为 $L+L_1$ 为顺时针方向，应用格林公式时需要加负号：
     $$\oint_{L+L_1} P dx + Q dy = -\iint_D \left( \frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y} \right) dxdy$$
     计算偏导数：
     $$\frac{\partial Q}{\partial x} = e^x \cos y, \quad \frac{\partial P}{\partial y} = e^x \cos y - m \implies \frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y} = m$$
     代入：
     $$\int_L = -\iint_D m dxdy = -m \cdot \text{Area}(D)$$
     由于 $D$ 是半径为 $a$ 的上半圆盘，面积为 $\frac{1}{2} \pi a^2$。
     * **结论**：$\int_L = -\frac{1}{2} m \pi a^2$。

### 例 2.1（路径无关与势函数）
**【题目】** 证明在整个 $xy$ 平面上，$(e^x \sin y - my) dx + (e^x \cos y - mx) dy$ 是某个函数的全微分，求这样一个函数，并计算其从 $O(0,0)$ 到 $B(1,1)$ 的曲线积分。
* **考点**：路径无关判定，原函数（势函数）求解。
* **解析**：
  1. **证明全微分性质**：
     设 $P = e^x \sin y - my$，$Q = e^x \cos y - mx$。
     $$\frac{\partial P}{\partial y} = e^x \cos y - m, \quad \frac{\partial Q}{\partial x} = e^x \cos y - m$$
     由于在全平面上 $\frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x}$ 恒成立，且全平面为单连通区域，所以该微分式是全微分，积分与路径无关。
  2. **求原函数 $u(x,y)$**：
     利用公式（取基准点 $(0,0)$）：
     $$u(x,y) = \int_0^x P(t,0) dt + \int_0^y Q(x, t) dt$$
     $$u(x,y) = \int_0^x (e^t \sin 0 - 0) dt + \int_0^y (e^x \cos t - mx) dt$$
     $$u(x,y) = 0 + \left[ e^x \sin t - mxt \right]_0^y = e^x \sin y - mxy + C$$
  3. **计算曲线积分**：
     由于积分与路径无关，可以直接用原函数在端点的值差计算：
     $$\int_{(0,0)}^{(1,1)} P dx + Q dy = u(1,1) - u(0,0) = (e^1 \sin 1 - m) - (e^0 \sin 0 - 0) = e\sin 1 - m$$

## 四、 PPT例题详解：高斯公式

### 例 3.2（球面三重积分的转换）
**【题目】** 计算 $\iint_{\Sigma} x^3 dydz + y^3 dzdx + z^3 dxdy$，其中 $\Sigma$ 为球面 $x^2+y^2+z^2=a^2$，方向取外侧。
* **考点**：高斯公式，球坐标三重积分。
* **解析**：
  1. **应用高斯公式**：
     球面 $\Sigma$ 包围的立体区域为 $\Omega: x^2+y^2+z^2 \le a^2$。
     $$\iint_{\Sigma} x^3 dydz + y^3 dzdx + z^3 dxdy = \iiint_{\Omega} \left( \frac{\partial (x^3)}{\partial x} + \frac{\partial (y^3)}{\partial y} + \frac{\partial (z^3)}{\partial z} \right) dV$$
     $$\iiint_{\Omega} 3(x^2+y^2+z^2) dV$$
  2. **使用球坐标计算三重积分**：
     引入球坐标：$x^2+y^2+z^2 = \rho^2$，体积微元 $dV = \rho^2 \sin\varphi d\rho d\varphi d\theta$，积分限为 $0 \le \rho \le a, 0 \le \varphi \le \pi, 0 \le \theta \le 2\pi$：
     $$I = 3 \int_0^{2\pi} d\theta \int_0^{\pi} \sin\varphi d\varphi \int_0^a \rho^2 \cdot \rho^2 d\rho$$
     $$I = 3 \cdot (2\pi) \cdot \left[ -\cos\varphi \right]_0^{\pi} \cdot \left[ \frac{\rho^5}{5} \right]_0^a = 6\pi \cdot 2 \cdot \frac{a^5}{5} = \frac{12}{5} \pi a^5$$

### 例 3.3（非封闭曲面高斯补面法）
**【题目】** 设流体速度场为 $\mathbf{v} = (x, y, z)$，求单位时间内流体流过抛物面 $\Sigma: y = x^2+z^2$ ($0 \le y \le h$) 且方向朝向外侧的通量。
* **考点**：非封闭曲面的高斯“补面法”。
* **解析**：
  1. **添加平面闭合区域**：
     抛物面 $\Sigma$ 在 $y=h$ 处开口，不是封闭曲面。我们添加一个朝向 $y$ 轴正向（外侧）的圆形盖子 $\Sigma_1: y=h, x^2+z^2 \le h$。
     此时封闭曲面 $\Sigma + \Sigma_1$ 围成了区域 $\Omega: x^2+z^2 \le y \le h$。
  2. **应用高斯公式**：
     由于 $\text{div} \mathbf{v} = 1+1+1 = 3$，有：
     $$\iint_{\Sigma} \mathbf{v} \cdot \mathbf{n} dS + \iint_{\Sigma_1} \mathbf{v} \cdot \mathbf{n} dS = \iiint_{\Omega} 3 dV = 3 V(\Omega)$$
     计算旋转体 $\Omega$ 的体积 $V(\Omega)$（用柱面坐标 $x=r\cos\theta, z=r\sin\theta, y=y$）：
     $$V(\Omega) = \int_0^{2\pi} d\theta \int_0^{\sqrt{h}} r dr \int_{r^2}^h dy = 2\pi \int_0^{\sqrt{h}} (h-r^2)r dr = 2\pi \left[ \frac{hr^2}{2} - \frac{r^4}{4} \right]_0^{\sqrt{h}} = \frac{\pi h^2}{2}$$
     所以，右端三重积分为 $3 V(\Omega) = \frac{3}{2} \pi h^2$。
  3. **计算在盖子 $\Sigma_1$ 上的通量**：
     在盖子 $\Sigma_1$ 上：$y = h$，单位外法向量为 $\vec{n} = (0, 1, 0)$。
     $$\iint_{\Sigma_1} \mathbf{v} \cdot \mathbf{n} dS = \iint_{\Sigma_1} y dS = h \cdot \text{Area}(\Sigma_1) = h \cdot (\pi h) = \pi h^2$$
  4. **求出所求曲面通量**：
     $$\iint_{\Sigma} = \frac{3}{2} \pi h^2 - \iint_{\Sigma_1} = \frac{3}{2} \pi h^2 - \pi h^2 = \frac{1}{2} \pi h^2$$

## 五、 PPT例题详解：斯托克斯公式

### 例 4.1（空间三角形边界的斯托克斯积分）
**【题目】** 计算 $I = \oint_{\Gamma} (y^2 - z^2) dx + (z^2 - x^2) dy + (x^2 - y^2) dz$，其中 $\Gamma$ 为平面 $x+y+z = 1$ 被三个坐标面所截三角形的边界，从 $z$ 轴正向看去为逆时针方向。
* **考点**：斯托克斯公式，空间曲线向曲面积分转化。
* **解析**：
  1. **求旋度向量**：
     被积项为 $\mathbf{F} = (y^2-z^2, z^2-x^2, x^2-y^2)$。其旋度为：
     $$\text{curl} \mathbf{F} = \left( \frac{\partial R}{\partial y} - \frac{\partial Q}{\partial z}, \frac{\partial P}{\partial z} - \frac{\partial R}{\partial x}, \frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y} \right) = -2(y+z, z+x, x+y)$$
  2. **应用斯托克斯公式**：
     由 $\Gamma$ 的逆时针方向，取的有向三角形曲面 $\Sigma$ 的法向量应指向外侧（即 $x+y+z=1$ 的正法向，三个分量均正）：
     $$\vec{n} = \frac{1}{\sqrt{3}} (1, 1, 1)$$
     代入公式：
     $$I = \iint_{\Sigma} \text{curl} \mathbf{F} \cdot \vec{n} dS = \iint_{\Sigma} -2(y+z + z+x + x+y) \frac{1}{\sqrt{3}} dS$$
     $$I = -\frac{4}{\sqrt{3}} \iint_{\Sigma} (x+y+z) dS$$
  3. **利用曲面代数关系化简**：
     因为在曲面 $\Sigma$ 上有恒等式 $x+y+z = 1$：
     $$I = -\frac{4}{\sqrt{3}} \iint_{\Sigma} 1 dS = -\frac{4}{\sqrt{3}} \text{Area}(\Sigma)$$
     该第一卦限截得的等边三角形边长为 $\sqrt{2}$，其面积为 $\text{Area}(\Sigma) = \frac{\sqrt{3}}{2}$。
     * **结论**：$I = -\frac{4}{\sqrt{3}} \cdot \frac{\sqrt{3}}{2} = -2$。

## 六、 备考解题套路与易错防范

### 1. 验证奇点的“挖洞法”
在使用格林公式或高斯公式时，**被积函数在区域内不能有分母为 0 的奇点**。
* 如果区域内包含奇点（如原点），必须以该奇点为圆心，在内部挖掉一个微小的圆盘/球体 $\Sigma_{\epsilon}$；
* 分开计算外边界和内边界的积分，其中内边界的方向与外边界相反。

### 2. 斯托克斯公式的右手法则
在应用斯托克斯公式时，曲线 $\Gamma$ 的方向与曲面 $\Sigma$ 的法向必须满足**右手法则**：四指弯曲方向为曲线方向，大拇指指向法向量。若方向反了，最终计算结果会差一个负号。
